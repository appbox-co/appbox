# Launch Week Day 1: নতুন Dashboard Live

আমরা বলেছিলাম Launch Week সোমবার শুরু হবে। আর Day 1 এখন live।

আজ আমরা নতুন Appbox dashboard experience ship করছি। Control panel জুড়ে এটি একটি বড় UI এবং workflow upgrade, এবং এর একটি সহজ লক্ষ্য আছে: প্রতিদিন আপনার appboxes চালানোকে আরও দ্রুত, পরিষ্কার এবং কম বিরক্তিকর করা।

এই update অনেক দিন ধরে চলছিল, এবং অবশেষে এটি আপনাদের হাতে দিতে পেরে আমি খুব খুশি।

- নতুন এবং সম্প্রতি updated apps

তাই tabs-এর মধ্যে লাফালাফি না করে, আপনি একটি page খুলেই সঙ্গে সঙ্গে গুরুত্বপূর্ণ জিনিস দেখতে পারবেন।

## Custom Domains এখন First Class

Custom domains এখন install experience-এর core অংশ, কোনো awkward extra step নয়।

কোনো app-এর domain দরকার হলে setup এখন সরাসরি install dialog-এ clear guided flow সহ হয়:

1. আপনার Appbox select করুন।
2. **Domain Type** বেছে নিন:
   - আপনার Appbox default domain ব্যবহার করতে **Appbox Domain**।
   - আপনার নিজস্ব domains-এর একটি ব্যবহার করতে **Custom Domain**।
3. **Base Domain** বেছে নিন।
4. আপনি যে **Subdomain** ব্যবহার করতে চান সেটি লিখুন।
5. আপনি custom domain select করলে, Appbox add করার exact DNS record দেখায় (host/name এবং target IP), তারপর সঙ্গে সঙ্গে verify করতে দেয়।

Records manually guess করা, docs copy করা, আর installs retry করার বদলে আপনি app install করার একই জায়গায় built-in DNS setup এবং verification step পাবেন।

## নতুন File Explorer (শূন্য থেকে তৈরি)

File explorer একেবারে শূন্য থেকে নতুনভাবে ভাবা হয়েছে। আগের experience-এ file management শুধু file delete করার মধ্যে সীমাবদ্ধ ছিল। এখন এটি একটি proper file manager, যেখানে আছে:

- uploads
- new folder creation
- rename
- delete
- download
- list/grid view switching
- breadcrumb navigation এবং contextual actions

তাই অতিরিক্ত app install না করেই আপনি native-ভাবে আপনার appbox files manage করতে পারবেন।

2026 সালে security optional হওয়া উচিত নয়। আজ থেকে Appbox accounts built-in two-factor authentication support করে।

## Dashboard জুড়ে নতুন Design Language

Navigation, spacing, cards, typography এবং interaction states জুড়ে visual system refresh করা হয়েছে। লক্ষ্য ছিল consistency এবং focus: কম visual noise, শক্তিশালী hierarchy, ভালো readability, এবং dark ও light theme দুটিতেই আরও modern product feel।

এখন আর মাঝরাতে apps install করার সময় আপনাকে চোখ-ধাঁধানো সাদা screen দেখতে হবে না!

## আরও ভালো Daily UX: Theme, Language, এবং দ্রুত Shortcuts

এই release-এ এমন quality-of-life features-ও আছে যা প্রতিদিন dashboard ব্যবহারকে আরও আরামদায়ক করে:

- **Native language switching** সরাসরি UI-তে (সময়ের সাথে আরও languages যোগ হবে)।
