# Launch Week Day 1: Το νέο Dashboard είναι διαθέσιμο

Είπαμε ότι το Launch Week ξεκινά τη Δευτέρα. Και η Day 1 είναι πλέον live.

Σήμερα διαθέτουμε τη νέα εμπειρία Appbox dashboard. Πρόκειται για μια σημαντική αναβάθμιση UI και ροής εργασίας σε όλο το control panel, με έναν απλό στόχο: να κάνει την καθημερινή διαχείριση των appboxes σας πιο γρήγορη, πιο ξεκάθαρη και λιγότερο ενοχλητική.

Αυτή η ενημέρωση βρισκόταν σε εξέλιξη για πολύ καιρό και είμαι πολύ χαρούμενος που μπορούμε επιτέλους να τη δώσουμε στα χέρια σας.

## Ένα νέο κέντρο ελέγχου

Πριν από σήμερα, αυτές οι πληροφορίες βρίσκονταν σε ξεχωριστές σελίδες και ροές. Σήμερα παρουσιάζουμε ένα ειδικό dashboard που τις συγκεντρώνει σε μία λειτουργική προβολή:

- τις πιο πρόσφατες ενημερώσεις blog και πλατφόρμας
- τις κάρτες επισκόπησης των υπηρεσιών σας
- τις ειδοποιήσεις σας
- τις νεότερες και πρόσφατα ενημερωμένες εφαρμογές

Έτσι, αντί να πηδάτε ανάμεσα σε καρτέλες, μπορείτε να ανοίξετε μία σελίδα και να δείτε αμέσως ό,τι έχει σημασία.

## Τα custom domains είναι πλέον βασικό στοιχείο

Τα custom domains είναι πλέον βασικό μέρος της εμπειρίας εγκατάστασης, όχι ένα άβολο επιπλέον βήμα.

Όταν μια εφαρμογή απαιτεί domain, η ρύθμιση γίνεται πλέον απευθείας στο παράθυρο εγκατάστασης με μια ξεκάθαρη καθοδηγούμενη ροή:

1. Επιλέξτε το Appbox σας.
2. Επιλέξτε **Domain Type**:
   - **Appbox Domain** για να χρησιμοποιήσετε το προεπιλεγμένο domain του Appbox σας.
   - **Custom Domain** για να χρησιμοποιήσετε ένα από τα δικά σας domains.
3. Επιλέξτε το **Base Domain**.
4. Εισαγάγετε το **Subdomain** που θέλετε να χρησιμοποιήσετε.
5. Αν επιλέξατε custom domain, το Appbox εμφανίζει την ακριβή DNS εγγραφή που πρέπει να προσθέσετε (host/name και target IP) και στη συνέχεια σας επιτρέπει να την επαληθεύσετε αμέσως.

Αντί να μαντεύετε χειροκίνητα εγγραφές, να αντιγράφετε τεκμηρίωση και να ξαναδοκιμάζετε εγκαταστάσεις, έχετε ενσωματωμένη ρύθμιση και επαλήθευση DNS στο ίδιο σημείο όπου εγκαθιστάτε την εφαρμογή.

## Νέος File Explorer (χτισμένος από την αρχή)

Ο file explorer έχει ξανασχεδιαστεί από την αρχή. Στην προηγούμενη εμπειρία, η διαχείριση αρχείων περιοριζόταν μόνο στη διαγραφή αρχείων. Τώρα είναι ένας κανονικός file manager με:

- uploads
- δημιουργία νέου φακέλου
- μετονομασία
- διαγραφή
- download
- εναλλαγή προβολής λίστας/πλέγματος
- breadcrumb navigation και ενέργειες με βάση τα συμφραζόμενα

Έτσι μπορείτε πραγματικά να διαχειρίζεστε τα αρχεία του appbox σας εγγενώς, αντί να εγκαθιστάτε πρόσθετη εφαρμογή.

## Έλεγχος ταυτότητας δύο παραγόντων

Η ασφάλεια δεν θα έπρεπε να είναι προαιρετική το 2026. Από σήμερα, οι λογαριασμοί Appbox υποστηρίζουν ενσωματωμένο έλεγχο ταυτότητας δύο παραγόντων.

Μπορείτε να ενεργοποιήσετε και να διαχειριστείτε το 2FA απευθείας από την περιοχή του λογαριασμού σας, συμπεριλαμβανομένης της διαχείρισης κωδικών ανάκτησης για ανάκτηση λογαριασμού.

## Νέα σχεδιαστική γλώσσα σε όλο το Dashboard

Το οπτικό σύστημα έχει ανανεωθεί σε navigation, αποστάσεις, κάρτες, τυπογραφία και καταστάσεις αλληλεπίδρασης. Ο στόχος ήταν συνέπεια και εστίαση: λιγότερος οπτικός θόρυβος, ισχυρότερη ιεραρχία, καλύτερη αναγνωσιμότητα και πιο σύγχρονη αίσθηση προϊόντος τόσο σε σκούρα όσο και σε φωτεινά θέματα.

Δεν θα έχετε πλέον μια εκτυφλωτική λευκή οθόνη όταν εγκαθιστάτε εφαρμογές τα μεσάνυχτα!

## Καλύτερη καθημερινή UX: θέμα, γλώσσα και ταχύτερες συντομεύσεις

- **Εγγενής αλλαγή γλώσσας** απευθείας στο UI (με περισσότερες γλώσσες να προστίθενται με τον χρόνο).
