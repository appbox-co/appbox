# Μια νέα εποχή αρχίζει στο Appbox

Μετά από μήνες δουλειάς πίσω από τα παρασκήνια, ξενύχτια και υπερβολικά πολύ καφέ, είμαι ενθουσιασμένος που επιτέλους σας δείχνω τι χτίζαμε. Αυτό δεν είναι απλώς άλλη μια ενημέρωση. Είναι η ανακατασκευή του Appbox από τα θεμέλια.

## Καλώς ήρθατε στο νέο μας ψηφιακό σπίτι

Βλέπετε το ολοκαίνουργιο website μας! Επιτέλους αποχαιρετήσαμε το παλιό WordPress setup. Όλο το πράγμα ξαναχτίστηκε με σύγχρονη τεχνολογία που είναι ταχύτερη, πιο ασφαλής και κλιμακώνεται πραγματικά σωστά.

Αυτό το νέο site θα γίνει το σπίτι για σωστή τεκμηρίωση, τακτικά blog posts (όπως αυτό) και τελικά το frontend για τη νέα μας πλατφόρμα. Ο σχεδιασμός δεν είναι απλώς πιο καθαρός - είναι πραγματικά λειτουργικός και κάνει την εύρεση πραγμάτων πολύ πιο εύκολη.

Χρησιμοποιούμε Next.js με ContentLayer στο εσωτερικό, κάτι που μας δίνει πολύ γρήγορη απόδοση και πολύ καλύτερη εμπειρία ανάπτυξης. Το site υποστηρίζει πολλαπλές γλώσσες, χειρίζεται αυτόματα SEO, έχει RSS feeds τόσο σε XML όσο και σε JSON και περιλαμβάνει dark mode επειδή κανείς δεν θα έπρεπε να αντιμετωπίζει φωτεινές λευκές οθόνες στις 2am.

## Γινόμαστε Open Source!

Γινόμαστε επίσης open source! Το website στο οποίο διαβάζετε αυτό το κείμενο; Ο κώδικας είναι δημόσιος και οποιοσδήποτε μπορεί να συνεισφέρει.

Αυτό είναι τεράστιο για εμάς. Πάντα προσπαθούσαμε να είμαστε διαφανείς, αλλά αυτό σημαίνει ότι κάνουμε πράξη τα λόγια μας και χτίζουμε πραγματικά με την κοινότητα, όχι απλώς για αυτή. Αν είστε developer και θέλατε ποτέ να βοηθήσετε στη διαμόρφωση των εργαλείων που χρησιμοποιείτε, τώρα μπορείτε. Δείτε το GitHub repo μας για οδηγίες συνεισφοράς και όλα όσα χρειάζεστε για να ξεκινήσετε.

## Παρουσιάζουμε το Appbox v2

Το νέο website είναι μόνο η αρχή. Δουλεύουμε πάνω στο Appbox v2, που είναι η μεγαλύτερη ενημέρωση που έχουμε κάνει ποτέ.

Το container deployment ήταν πάντα το δικό μας πεδίο, αλλά το v2 το προχωρά περισσότερο με αυτό που ζητούσαν όλοι, πλήρη υποστήριξη VPS. Τα VPS apps θα σας δίνουν πλήρες root access χωρίς περιορισμούς. Αν χρειάζεστε απόλυτο έλεγχο στο περιβάλλον σας, αυτό περιμένατε.

Το έχουμε ήδη σε λειτουργία σε έναν server και δουλεύει πολύ καλά. Θα μπορείτε να εγκαθιστάτε VPS apps όπως οποιαδήποτε άλλη εφαρμογή από το app store. Αυτό θα σας επιτρέψει να εγκαθιστάτε εφαρμογές που δεν είναι σήμερα δυνατές, όπως Docker, Snap και Tailscale. Απαιτεί OS update στους servers μας σε Debian 13, επειδή χρησιμοποιούμε μερικές νέες δυνατότητες virtualization για να το κάνουμε πιο ασφαλές και να αυξήσουμε την απόδοση. Θα χρειαστεί να διαθέσουμε αυτή την ενημέρωση πριν μπορέσουμε να λανσάρουμε τη λειτουργία, αλλά θα υπάρξουν περισσότερες ενημερώσεις σύντομα.

Αυτό συνδυάζει την αποδεδειγμένη container infrastructure μας με πραγματική ευελιξία VPS, δίνοντάς σας το καλύτερο και από τους δύο κόσμους.

## Χτίζουμε τη δική μας δικτυακή υποδομή

Να τα πραγματικά μεγάλα νέα, χτίζουμε το δικό μας δίκτυο.

Όχι απλώς προσθέτοντας περισσότερους servers ή αναβαθμίζοντας bandwidth. Έχουμε το δικό μας Autonomous System Number ([AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)) και το δικό μας IP block (79.140.195.0/24) για να χτίσουμε ένα δίκτυο που είναι εξ ολοκλήρου δικό μας. Ουσιαστικά, γινόμαστε ο δικός μας ISP με τη δική μας επίσημη ταυτότητα στο internet και δικό μας address space, ώστε να μπορούμε να ελέγχουμε πώς ταξιδεύουν τα δεδομένα σας στο internet [1].

Τι σημαίνει αυτό για εσάς; Μπορούμε να ελέγχουμε τα πάντα, να τα βελτιστοποιούμε ειδικά για τους χρήστες Appbox και να προσφέρουμε αξιοπιστία και απόδοση που πριν δεν ήταν δυνατές.

## Peering με AMS-IX

Για να συνοδεύσουμε την ανάπτυξη του δικτύου μας, μόλις έχουμε το network setup, θα αρχίσουμε peering με AMS-IX για 100G connectivity. Αυτό μας δίνει άμεση πρόσβαση σε ένα από τα μεγαλύτερα internet exchanges του κόσμου, που σημαίνει χαμηλότερο latency και καλύτερες ταχύτητες για όλους. Σκεφτείτε το σαν να έχετε έναν άμεσο αυτοκινητόδρομο προς τους μεγάλους internet providers αντί να παίρνετε τους παράδρομους [2].

Αυτό το είδος connectivity είναι συνήθως διαθέσιμο μόνο σε πολύ μεγαλύτερες εταιρείες, αλλά θέλουμε να παρέχουμε enterprise-grade απόδοση ανεξάρτητα από το πλάνο σας.

Έχουμε πλέον τακτοποιήσει τη συμφωνία για την αγορά hardware, οπότε είμαστε έτοιμοι. Θα παραλάβουμε το νέο hardware το αργότερο έως τον October, πιθανώς ήδη από τον September. Αυτό σημαίνει ότι θα πρέπει να είμαστε σε λειτουργία στο νέο δίκτυο λίγο αργότερα.

## Προσαρμογές τιμών

Αυτό είναι το κομμάτι που πραγματικά δεν θέλαμε να χρειαστεί να γράψουμε. Το να χτίζεις υποδομή σαν αυτή είναι ακριβό, και παρότι αναλαμβάνουμε οι ίδιοι το μεγαλύτερο μέρος του κόστους, δυστυχώς χρειάζεται να κάνουμε μια μικρή προσαρμογή στις τιμές μας τόσο για νέους όσο και για υπάρχοντες πελάτες.

Θέλω να είμαι ειλικρινής γι' αυτό. Η νέα τιμολόγηση είναι ήδη live στο site και θα επικοινωνήσουμε σύντομα με τους υπάρχοντες πελάτες με λεπτομέρειες για το πώς τους επηρεάζει.

Ξέρουμε πόσο πιεσμένοι είναι όλοι αυτή τη στιγμή, και ειλικρινά αυτή ήταν μία από τις πιο δύσκολες αποφάσεις που χρειάστηκε να πάρουμε. Κρατήσαμε τις αυξήσεις όσο πιο μικρές μπορούσαμε - ουσιαστικά καλύπτουμε τα πάντα εκτός από το απόλυτο ελάχιστο που χρειαζόμαστε για να λειτουργήσει αυτό.

Θα κάνουμε όμως αυτή την αύξηση τιμής περισσότερο από άξια. Θα παρατηρήσετε πραγματικά τη διαφορά στην απόδοση και την αξιοπιστία. Επιπλέον, αυτό σημαίνει ότι μπορούμε να συνεχίσουμε να επεκτείνουμε το δίκτυό μας και να προσθέτουμε περισσότερο peering στο μέλλον, οπότε τα πράγματα μόνο θα βελτιώνονται από εδώ και πέρα.

Λανσάραμε αυτές τις ενημερωμένες τιμές για νέους πελάτες πριν από 3 μήνες, και τα σχόλια ήταν καλά, οπότε νιώθουμε ότι έχουμε βρει καλή ισορροπία ανάμεσα στο να κρατάμε τα πράγματα προσιτά και να χρηματοδοτούμε τις βελτιώσεις που αξίζετε, παραμένοντας πολύ ανταγωνιστικοί. Για παράδειγμα, το δημοφιλές πλάνο NG-4500 αυξάνεται μόνο κατά €2.08 τον μήνα, κάτι που εξακολουθεί να είναι πολύ λιγότερο από συγκρίσιμα πλάνα ανταγωνιστών.

Για υπάρχοντες πελάτες, θα πληρώνετε μόνο τη διαφορά ανάμεσα στην παλιά και τη νέα τιμή. Αν έχετε οποιεσδήποτε promotional discounts, παραμένουν επιπλέον της νέας βασικής τιμής. Εκτιμούμε την αφοσίωσή σας, ειδικά σε εποχές σαν αυτές.

Η δομή εκπτώσεών μας παραμένει η ίδια:
- 5% έκπτωση για τριμηνιαίες πληρωμές
- 10% έκπτωση για πληρωμές 6 μηνών
- 15% έκπτωση για ετήσιες πληρωμές

Αν έχετε ρυθμίσει αυτόματες πληρωμές, η προσαρμογή θα γίνει στον επόμενο κύκλο χρέωσης μετά τη λήψη του email. Δεν χρειάζεται να κάνετε τίποτα.

## Περισσότερος αποθηκευτικός χώρος!

Δεν αφορά όμως μόνο το να πληρώνετε περισσότερα. Αυξάνουμε επίσης τον αποθηκευτικό χώρο στο χαμηλότερο πλάνο μας από 3TB σε 3.5TB. Αυτό είναι αύξηση 16%, που κάνει το entry-level πλάνο μας ακόμη καλύτερη επιλογή παρά την αλλαγή τιμής.

## Τι ακολουθεί;

Αυτό είναι μόνο η αρχή όσων έρχονται για το Appbox. Τις επόμενες εβδομάδες και μήνες θα βλέπετε τακτικές ενημερώσεις καθώς διαθέτουμε νέες λειτουργίες, βελτιώνουμε τις υπάρχουσες και συνεχίζουμε να χτίζουμε την υποδομή για την επόμενη γενιά app hosting.

Σχεδιάζουμε να αρχίσουμε να δοκιμάζουμε τη νέα δικτυακή υποδομή μας τον September/October και θα σας κρατάμε ενήμερους καθώς προχωράμε.

Ο roadmap μας περιλαμβάνει:
- VPS apps που δεν θα έχουν περιορισμούς (όπως Docker, Snap και Tailscale)
- Καλύτερη τεκμηρίωση που πραγματικά καλύπτει τα πάντα
- Ένα community showcase για ωραία πράγματα που χτίζουν οι άνθρωποι με το Appbox
- Τακτικά τεχνικά blog posts για την υποδομή μας
- Νέες containerized apps για συγκεκριμένες περιπτώσεις χρήσης
- Καλύτερα εργαλεία monitoring και analytics
- Ενισχυμένα χαρακτηριστικά ασφαλείας σε όλα τα πλάνα

Ευχαριστούμε που μένετε μαζί μας. Χωρίς την κοινότητά μας, τίποτα από αυτά δεν θα ήταν δυνατό.

Περισσότερες ενημερώσεις σύντομα.
