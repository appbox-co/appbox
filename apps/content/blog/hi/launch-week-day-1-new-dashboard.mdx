## Custom Domains अब First Class हैं

Custom domains अब install experience का core हिस्सा हैं, कोई awkward extra step नहीं।

जब किसी app को domain चाहिए, setup अब install dialog में directly clear guided flow के साथ होता है:

1. अपना Appbox चुनें।
2. **Domain Type** चुनें:
   - अपने Appbox default domain के लिए **Appbox Domain**।
   - अपने domains में से किसी एक के लिए **Custom Domain**।
3. **Base Domain** चुनें।
4. वह **Subdomain** दर्ज करें जिसे आप इस्तेमाल करना चाहते हैं।
5. अगर आपने custom domain चुना है, तो Appbox add करने के लिए exact DNS record दिखाता है (host/name और target IP), फिर उसे immediately verify करने देता है।

Records manually guess करने, docs copy करने और installs retry करने के बजाय, app install करने वाली जगह पर ही built-in DNS setup और verification step मिलता है।

## नया File Explorer (Scratch से बनाया गया)

File explorer को scratch से फिर से imagine किया गया है। Previous experience में file management सिर्फ files delete करने तक limited था। अब यह proper file manager है जिसमें:

- uploads
- new folder creation
- rename
- delete
- download
- list/grid view switching
- breadcrumb navigation और contextual actions

इसलिए आप additional app install किए बिना अपने appbox files natively manage कर सकते हैं।
