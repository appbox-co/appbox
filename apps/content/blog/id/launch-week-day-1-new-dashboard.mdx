# Launch Week Day 1: Dashboard Baru Sudah Live

## Custom Domain Kini Menjadi Bagian Utama

Custom domain sekarang menjadi bagian inti dari pengalaman instalasi, bukan langkah tambahan yang canggung.

Saat aplikasi membutuhkan domain, setup kini terjadi langsung di dialog instalasi dengan flow terpandu yang jelas:

1. Pilih Appbox Anda.
2. Pilih **Domain Type**:
   - **Appbox Domain** untuk menggunakan domain default Appbox Anda.
   - **Custom Domain** untuk menggunakan salah satu domain Anda sendiri.
3. Pilih **Base Domain**.
4. Masukkan **Subdomain** yang ingin Anda gunakan.
5. Jika Anda memilih custom domain, Appbox menampilkan record DNS persis yang perlu ditambahkan (host/name dan target IP), lalu memungkinkan Anda memverifikasinya langsung.

Alih-alih menebak record secara manual, menyalin dokumentasi, dan mencoba ulang instalasi, Anda mendapatkan setup dan verifikasi DNS bawaan di tempat yang sama tempat Anda menginstal aplikasi.

## File Explorer Baru (Dibangun dari Nol)

File explorer telah dibayangkan ulang dari nol. Pada pengalaman sebelumnya, manajemen file hanya terbatas pada menghapus file. Sekarang ini adalah file manager yang layak dengan:

- upload
- pembuatan folder baru
- ganti nama
- hapus
- download
- pergantian tampilan list/grid
- navigasi breadcrumb dan tindakan kontekstual

Jadi Anda benar-benar dapat mengelola file appbox secara native tanpa menginstal aplikasi tambahan.

## Bahasa Desain Baru di Seluruh Dashboard

Sistem visual telah diperbarui di seluruh navigasi, spacing, kartu, tipografi, dan state interaksi. Targetnya adalah konsistensi dan fokus: lebih sedikit noise visual, hierarki lebih kuat, keterbacaan lebih baik, dan rasa produk yang lebih modern baik di tema gelap maupun terang.
