# OpenClaw Kini Ada di Appbox

Kami baru saja menambahkan sesuatu ke app store yang secara pribadi sudah lama membuat saya antusias.

[OpenClaw](https://openclaw.ai) adalah asisten AI open-source yang berjalan sepenuhnya di infrastruktur Anda sendiri. Anggap saja sebagai alternatif privat dan self-hosted untuk asisten AI cloud, tetapi yang satu ini tinggal di Appbox Anda, mengingat percakapan Anda, dan benar-benar dapat melakukan sesuatu untuk Anda.

Anda berbicara dengannya melalui aplikasi chat yang sudah Anda gunakan. WhatsApp, Telegram, Discord, Slack, Signal, bahkan iMessage. Ia merespons dengan kekuatan penuh model dari Anthropic, OpenAI, Google, xAI, Mistral, dan lainnya. Anda membawa API key sendiri, memilih model, dan semuanya tetap berada di server Anda.

Itu saja sudah berguna. Tetapi alasan saya menulis posting blog tentang ini, bukan hanya diam-diam menambahkannya ke store, adalah apa yang terjadi saat Anda menggabungkannya dengan bagian lain dari Appbox Anda.

## Ia Dapat Melihat Aplikasi Anda yang Lain

Inilah yang membuat OpenClaw di Appbox berbeda dari menjalankannya di tempat lain.

Setiap Appbox memiliki lapisan data bersama. Setiap aplikasi terinstal mendapat direktori di bawah mount umum, dan ada area penyimpanan bersama yang dapat dibaca dan ditulis oleh aplikasi mana pun. OpenClaw memiliki akses ke filesystem bersama yang sama. Artinya ia dapat melihat data dari aplikasi terinstal lain dan penyimpanan umum Anda, bukan melalui integrasi API yang rapuh, tetapi melalui akses filesystem langsung.

Dalam praktiknya, asisten AI Anda tidak terisolasi di container sendiri tanpa konteks. Ia berada tepat di sebelah file, database, media, kode, dan konfigurasi Anda.

## Ia Dapat Login ke Aplikasi Anda yang Lain

Selain filesystem bersama, OpenClaw memiliki browser headless penuh (Chromium dengan Puppeteer) dan setiap klien database umum serta alat CLI bawaan. Jika Anda memberinya detail login untuk salah satu aplikasi lain, ia benar-benar dapat masuk dan menggunakannya atas nama Anda.

Ingin memperbarui dokumentasi tim Anda di Docmost? Beri tahu OpenClaw apa yang harus ditulis dan berikan URL serta kredensialnya. Ia akan login, menavigasi ke halaman yang tepat, dan melakukan edit. Butuh landing page baru di stack Nginx & PHP Anda? Jelaskan apa yang Anda inginkan dan OpenClaw dapat menulis HTML, CSS, dan JavaScript, lalu men-deploy file langsung ke web root Anda melalui filesystem bersama. Menjalankan WordPress? Minta ia membuat draft dan menerbitkan posting blog, memperbarui pengaturan plugin, atau mengubah gaya halaman. Ia dapat mengoperasikan dashboard admin seperti Anda.

Ini juga tidak terbatas pada UI web. OpenClaw hadir dengan `mysql`, `psql`, `redis-cli`, `mongosh`, dan dukungan SSH, sehingga dapat terhubung langsung ke database dan layanan Anda menggunakan kredensial apa pun yang Anda berikan. Poin utamanya adalah OpenClaw tidak memerlukan integrasi atau plugin khusus untuk setiap aplikasi. Jika Anda bisa login ke sana, asisten Anda juga bisa.

## Seperti Apa Bentuknya

Mari saya jelaskan beberapa contoh yang lebih spesifik tentang apa yang dapat Anda lakukan saat asisten AI memiliki akses ke ekosistem Appbox Anda.

### Perawatan Media Server

Jika Anda menjalankan Plex, Jellyfin, atau Emby, OpenClaw hadir dengan `ffmpeg`, `mediainfo`, dan `exiftool` bawaan. Ini berguna untuk memeriksa metadata file atau menemukan masalah encoding tanpa harus menelusuri direktori secara manual. Minta ia mencantumkan isi folder, memeriksa format, atau membantu merapikan konvensi penamaan agar media server Anda mengindeks semuanya dengan benar.

### Pekerjaan Database Tanpa Repot

Container OpenClaw dilengkapi `mysql`, `psql`, `sqlite3`, `redis-cli`, dan `mongosh` yang sudah terinstal. Jika Anda menjalankan MariaDB, Anda dapat meminta OpenClaw mengkueri database, memeriksa schema, men-debug query lambat, atau membantu menulis migration. Semuanya melalui pesan chat di ponsel Anda.

Menjalankan WordPress atau CMS lain yang didukung MariaDB? Minta OpenClaw memeriksa kesehatan database, menemukan metadata posting orphaned, atau mengekspor konten tertentu. Anda tidak perlu SSH ke apa pun atau mengingat flag CLI yang benar. Cukup jelaskan kebutuhan Anda dengan bahasa biasa.

### GitLab dan Workflow Kode

Jika Anda menginstal GitLab, OpenClaw dapat bekerja langsung dengan repository Anda. Ia memiliki `git` bawaan bersama `ripgrep` dan `fd` untuk pencarian kode cepat. Minta ia mencari pola tertentu di seluruh repo, merangkum commit terbaru, atau membantu memahami kode yang belum familiar.

Untuk tim yang menggunakan GitLab untuk project internal, memiliki asisten AI yang benar-benar dapat membaca codebase Anda dan menjawab pertanyaan tentangnya tanpa mengirim kode ke layanan cloud pihak ketiga adalah peningkatan bermakna untuk workflow pengembangan.

### Organisasi File dan Sinkronisasi

Menjalankan Nextcloud, SFTPGo, atau File Browser? OpenClaw dapat membantu mengelola file yang mengalir melalui layanan tersebut. Ia dapat mencari dokumen, menata ulang struktur direktori, menemukan duplikat, dan memindahkan item antara penyimpanan bersama dan direktori khusus aplikasi.

Ia juga memiliki `rclone`, jadi jika Anda perlu menyiapkan job sinkronisasi atau memindahkan data antar provider penyimpanan cloud, Anda dapat menjelaskan apa yang diinginkan dan membiarkan OpenClaw menangani konfigurasinya.

### Monitoring dan Diagnostik

Jika Anda menginstal Uptime Kuma untuk memonitor layanan, OpenClaw memberi Anda lapisan percakapan di atas setup monitoring Anda. Selain sekadar menerima alert, Anda dapat meminta OpenClaw menjalankan diagnostik jaringan. Ia hadir dengan `ping`, `dig`, `nslookup`, `traceroute`, `mtr`, bahkan `nmap` dan `tcpdump` untuk investigasi lebih dalam.

Ada sesuatu yang down? Kirim pesan ke asisten Anda, minta ia memeriksa apa yang terjadi, dan dapatkan penjelasan bahasa biasa alih-alih menatap dashboard.

### Library Musik dan Audiobook

Pengguna Navidrome dan audiobookshelf dapat memanfaatkan tooling media OpenClaw. Minta ia memindai library musik Anda untuk file dengan tag metadata yang hilang atau salah, mengidentifikasi track yang mungkin perlu re-encoding, atau membantu mengatur chapter audiobook yang tidak terpisah dengan benar. Kombinasi `ffmpeg` dan `exiftool` menangani sebagian besar yang Anda perlukan.

### Jupyter dan Pekerjaan Data

Jika Anda menggunakan Jupyter Notebook untuk analisis data, OpenClaw melengkapinya dengan baik. Ia memiliki Python 3 dengan dukungan pip dan venv, sehingga dapat membantu menulis script, memproses file data di penyimpanan bersama Anda, dan bahkan menjalankan analisis cepat secara langsung. Anggap saja sebagai antarmuka percakapan ke data yang sama yang digunakan notebook Anda.

### Infrastructure as Code

Untuk yang lebih teknis dan suka bereksperimen, OpenClaw hadir dengan CLI `docker`, `kubectl`, `terraform`, dan `ansible`. Jika Anda menjalankan VPS di Appbox dan mengelola infrastruktur, Anda dapat menggunakan OpenClaw sebagai asisten ops percakapan. Jelaskan apa yang ingin di-deploy dan biarkan ia membantu menghasilkan serta menerapkan konfigurasi.

## Memori Persisten

Salah satu hal yang membedakan OpenClaw dari sekadar jendela chat ke model AI adalah ia mengingat Anda. Ia membangun konteks dari waktu ke waktu: preferensi Anda, cara Anda suka mengatur sesuatu, project yang sedang Anda kerjakan, konvensi penamaan yang Anda gunakan. Semakin banyak Anda berinteraksi dengannya, semakin berguna ia.

Ini sangat kuat di Appbox karena konteksnya bukan hanya percakapan. Konteks itu berakar pada environment Anda yang sebenarnya. File Anda, aplikasi Anda, data Anda.

## Skill yang Sudah Dimuat

OpenClaw memiliki sistem skill dengan 50+ ekstensi buatan komunitas yang tersedia di [ClawHub](https://clawhub.ai/skills). Skill memberi agent kemampuan khusus untuk aplikasi, alat, atau workflow tertentu.

Di Appbox, kami tidak hanya memberi agent kosong. Image dikirim dengan set skill kurasi yang sudah terinstal sehingga OpenClaw memahami environment Anda sejak awal:

- **Kesadaran platform**: mengetahui layout data bersama Appbox, alat CLI yang tersedia, dan cara mengarahkan Anda ke dukungan atau upgrade VPS saat dibutuhkan
- **Skill khusus aplikasi**: sudah dimuat untuk WordPress, Nginx, GitLab, Nextcloud, Emby, MariaDB, Tautulli, Mautic, WireGuard, dan Rocket.Chat, sehingga agent sudah tahu cara bekerja dengan aplikasi ini saat Anda menginstalnya
- **Esensial SSH**: untuk terhubung ke VPS atau server remote langsung dari agent
- **Otomasi browser (Playwright MCP)**: mendukung kemampuan untuk login dan mengoperasikan aplikasi lain melalui UI web mereka
- **Pencarian web dan lookup dokumentasi**: pencarian multi-engine dan ClawDDocs untuk mengambil materi referensi saat dibutuhkan
- **Self-improvement**: skill untuk menemukan skill baru di ClawHub dan memeriksanya sebelum instalasi

Anda dapat menginstal skill tambahan kapan saja. Cukup temukan satu di [ClawHub](https://clawhub.ai/skills), salin link, dan tempel ke agent Anda. Ia akan mengunduh dan menginstal skill tersebut sendiri.

## Mulai

Instal OpenClaw dari Appbox app store seperti aplikasi lainnya. Ia membutuhkan dua slot.

Setelah instalasi, Anda akan melihat Gateway Token di halaman detail aplikasi. Itu adalah login dashboard Anda. Buka URL dashboard, masukkan token, dan Anda masuk.

Dari sana, satu hal yang perlu Anda konfigurasi adalah provider model AI Anda. Tambahkan API key dari provider mana pun yang Anda pilih. OpenRouter adalah titik awal termudah karena satu key memberi Anda akses ke banyak model. Kami telah menulis [panduan setup lengkap](/docs/apps/openclaw-model-provider-setup) yang membahas cara mendapatkan API key dari OpenRouter, OpenAI, Anthropic, Google, xAI, Mistral, dan lainnya.

Setelah model dikonfigurasi, Anda dapat mulai chat dengan asisten langsung dari dashboard. Jika Anda ingin berbicara dengannya melalui WhatsApp, Telegram, Discord, atau channel lain, itu juga dikonfigurasi melalui dashboard.

## Mengapa Ini Penting

Ada banyak antarmuka chat AI di luar sana. Sebagian besar adalah layanan cloud tempat data Anda meninggalkan kontrol Anda begitu Anda mengirim pesan. OpenClaw berbeda karena semuanya tetap berada di Appbox Anda. Asisten, memorinya, percakapan Anda, dan data yang dikerjakannya.

Dan karena ia berada berdampingan dengan aplikasi lain Anda dengan akses filesystem bersama, ia benar-benar dapat melakukan hal-hal di seluruh stack self-hosted Anda, bukan hanya menjawab pertanyaan tentangnya.

Kami benar-benar penasaran melihat apa yang akan dibangun orang dengan ini. Jika Anda menyiapkan workflow menarik atau menemukan cara kreatif untuk menghubungkan OpenClaw dengan aplikasi lain, kami ingin mendengarnya.
