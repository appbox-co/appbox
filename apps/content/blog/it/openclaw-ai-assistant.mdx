# OpenClaw è ora su Appbox

Abbiamo appena aggiunto all'app store qualcosa che personalmente mi entusiasma da un po'.

[OpenClaw](https://openclaw.ai) è un assistente AI open-source che gira interamente sulla tua infrastruttura. Pensalo come un'alternativa privata e self-hosted agli assistenti AI cloud, con la differenza che vive sul tuo Appbox, ricorda le tue conversazioni e può davvero fare cose per te.

Gli parli tramite le app di chat che già usi. WhatsApp, Telegram, Discord, Slack, Signal, persino iMessage. Risponde con tutta la potenza dei modelli di Anthropic, OpenAI, Google, xAI, Mistral e altri. Porti la tua API key, scegli il tuo modello e tutto resta sul tuo server.

Già questo sarebbe utile. Ma il motivo per cui sto scrivendo un post sul blog invece di aggiungerlo semplicemente allo store in silenzio è ciò che succede quando lo combini con il resto del tuo Appbox.

## Può vedere le tue altre app

Ecco cosa rende OpenClaw su Appbox diverso dall'eseguirlo altrove.

Ogni Appbox ha un livello dati condiviso. Le app installate ricevono ciascuna una directory sotto un mount comune, e c'è un'area di storage condivisa che qualsiasi app può leggere e scrivere. OpenClaw ha accesso allo stesso filesystem condiviso. Questo significa che può vedere i dati delle altre app installate e del tuo storage generale, non tramite qualche fragile integrazione API, ma tramite accesso diretto al filesystem.

In pratica, il tuo assistente AI non è isolato nel proprio container senza contesto. È seduto proprio accanto ai tuoi file, database, media, codice e configurazioni.

## Può accedere alle tue altre app

Oltre al filesystem condiviso, OpenClaw include un browser headless completo (Chromium con Puppeteer) e ogni client database e strumento CLI comune già integrato. Se gli fornisci i dettagli di login di una delle tue altre app, può davvero accedere e usarla per tuo conto.

Vuoi aggiornare la documentazione del team su Docmost? Di' a OpenClaw cosa scrivere e dagli URL e credenziali. Accederà, navigherà alla pagina corretta e farà le modifiche. Ti serve una nuova landing page sul tuo stack Nginx & PHP? Descrivi cosa vuoi e OpenClaw può scrivere HTML, CSS e JavaScript, poi distribuire i file direttamente nella web root attraverso il filesystem condiviso. Usi WordPress? Chiedigli di preparare e pubblicare un post, aggiornare le impostazioni di un plugin o ritoccare lo stile di una pagina. Può guidare la dashboard di amministrazione proprio come faresti tu.

Non si limita nemmeno alle UI web. OpenClaw include `mysql`, `psql`, `redis-cli`, `mongosh` e supporto SSH, quindi può connettersi direttamente ai tuoi database e servizi usando qualunque credenziale tu fornisca. Il punto chiave è che OpenClaw non ha bisogno di integrazioni o plugin speciali per ogni app. Se puoi accedervi tu, può farlo anche il tuo assistente.

## Come appare nella pratica

Vediamo alcuni esempi più specifici di cosa puoi fare quando un assistente AI ha accesso al tuo ecosistema Appbox.

### Manutenzione dei media server

Se esegui Plex, Jellyfin o Emby, OpenClaw include `ffmpeg`, `mediainfo` ed `exiftool`. Sono utili per controllare i metadati dei file o individuare problemi di encoding senza dover scavare manualmente nelle directory. Chiedigli di elencare cosa c'è in una cartella, controllare formati o aiutarti a sistemare le convenzioni di naming in modo che il tuo media server indicizzi tutto correttamente.

### Lavoro sui database senza dolore

Il container di OpenClaw arriva con `mysql`, `psql`, `sqlite3`, `redis-cli` e `mongosh` preinstallati. Se esegui MariaDB, puoi chiedere a OpenClaw di interrogare i database, ispezionare schemi, fare debug di query lente o aiutarti a scrivere migrazioni. Tutto tramite un messaggio chat dal telefono.

Esegui WordPress o un altro CMS basato su MariaDB? Chiedi a OpenClaw di controllare la salute del database, trovare metadati post orfani o esportare contenuti specifici. Non devi entrare in SSH né ricordare i flag CLI corretti. Descrivi semplicemente ciò che ti serve in linguaggio naturale.

### GitLab e workflow di codice

Se hai GitLab installato, OpenClaw può lavorare direttamente con i tuoi repository. Ha `git` integrato insieme a `ripgrep` e `fd` per una ricerca rapida nel codice. Chiedigli di cercare nei repo un pattern specifico, riassumere i commit recenti o aiutarti a capire codice non familiare.

Per i team che usano GitLab per progetti interni, avere un assistente AI che può davvero leggere il codice e rispondere a domande senza inviare il codice a un servizio cloud di terze parti è un miglioramento significativo del workflow di sviluppo.

### Organizzazione e sync dei file

Esegui Nextcloud, SFTPGo o File Browser? OpenClaw può aiutare a gestire i file che passano attraverso quei servizi. Può cercare nei documenti, riorganizzare strutture di directory, trovare duplicati e spostare elementi tra lo storage condiviso e le directory specifiche delle app.

Ha anche `rclone` disponibile, quindi se devi configurare job di sync o spostare dati tra provider di cloud storage, puoi descrivere cosa vuoi e lasciare che OpenClaw gestisca la configurazione.

### Monitoraggio e diagnostica

Se hai installato Uptime Kuma per monitorare i tuoi servizi, OpenClaw aggiunge un livello conversazionale sopra la tua configurazione di monitoraggio. Oltre a ricevere avvisi, puoi chiedere a OpenClaw di eseguire diagnostica di rete. Include `ping`, `dig`, `nslookup`, `traceroute`, `mtr` e persino `nmap` e `tcpdump` per indagini più profonde.

Qualcosa va giù? Scrivi al tuo assistente, chiedigli di controllare cosa sta succedendo e ottieni una spiegazione in linguaggio naturale invece di fissare dashboard.

### Librerie musicali e audiolibri

Gli utenti Navidrome e audiobookshelf possono beneficiare degli strumenti media di OpenClaw. Chiedigli di scansionare la tua libreria musicale alla ricerca di file con tag metadati mancanti o errati, identificare tracce che potrebbero dover essere ricodificate o aiutarti a organizzare capitoli di audiolibri che non sono stati divisi correttamente. La combinazione `ffmpeg` ed `exiftool` gestisce gran parte di ciò che serve.

### Jupyter e lavoro sui dati

Se usi Jupyter Notebook per l'analisi dati, OpenClaw lo completa bene. Ha Python 3 con supporto pip e venv, quindi può aiutarti a scrivere script, elaborare file dati nello storage condiviso e persino eseguire analisi rapide direttamente. Pensalo come un'interfaccia conversazionale sugli stessi dati con cui lavorano i tuoi notebook.

### Infrastructure as Code

Per i più tecnicamente avventurosi, OpenClaw include le CLI `docker`, `kubectl`, `terraform` e `ansible`. Se esegui una VPS su Appbox e gestisci infrastruttura, puoi usare OpenClaw come assistente ops conversazionale. Descrivi ciò che vuoi distribuire e lascia che ti aiuti a generare e applicare configurazioni.

## Memoria persistente

Una delle cose che distingue OpenClaw da una semplice finestra chat verso un modello AI è che si ricorda di te. Costruisce contesto nel tempo: le tue preferenze, il modo in cui ti piace organizzare le cose, i progetti su cui lavori, le convenzioni di naming che usi. Più interagisci con lui, più diventa utile.

Su Appbox questo è particolarmente potente perché il contesto non è solo conversazionale. È radicato nel tuo ambiente reale. I tuoi file, le tue app, i tuoi dati.

## Skill preinstallate

OpenClaw ha un sistema di skill con oltre 50 estensioni create dalla community disponibili su [ClawHub](https://clawhub.ai/skills). Le skill danno all'agent capacità specializzate per app, strumenti o workflow specifici.

Su Appbox non ti diamo solo l'agent base. L'immagine include già un set curato di skill installate, così OpenClaw capisce il tuo ambiente fin da subito:

- **Consapevolezza della piattaforma**: conosce il layout dati condiviso di Appbox, gli strumenti CLI disponibili e come guidarti verso supporto o upgrade VPS quando necessario
- **Skill specifiche per app**: preinstallate per WordPress, Nginx, GitLab, Nextcloud, Emby, MariaDB, Tautulli, Mautic, WireGuard e Rocket.Chat, così l'agent sa già come lavorare con queste app quando le hai installate
- **Essenziali SSH**: per collegarsi alle tue VPS o a server remoti direttamente dall'agent
- **Automazione browser (Playwright MCP)**: alimenta la capacità di accedere e guidare le altre app tramite le loro UI web
- **Ricerca web e consultazione documentazione**: ricerca multi-engine e ClawDDocs per recuperare materiale di riferimento al volo
- **Auto-miglioramento**: skill per scoprire nuove skill su ClawHub e valutarle prima dell'installazione

Puoi installare skill aggiuntive in qualsiasi momento. Trovane una su [ClawHub](https://clawhub.ai/skills), copia il link e incollalo nel tuo agent. La scaricherà e installerà da solo.

## Come iniziare

Installa OpenClaw dall'app store Appbox come qualsiasi altra app. Richiede due slot.

Dopo l'installazione, vedrai un Gateway Token nella pagina di dettaglio dell'app. È il tuo login alla dashboard. Apri l'URL della dashboard, inserisci il token e sei dentro.

Da lì, l'unica cosa da configurare è il provider del modello AI. Aggiungi una API key dal provider che preferisci. OpenRouter è il punto di partenza più semplice perché una sola chiave ti dà accesso a molti modelli. Abbiamo scritto una [guida completa alla configurazione](/docs/apps/openclaw-model-provider-setup) che illustra come ottenere API key da OpenRouter, OpenAI, Anthropic, Google, xAI, Mistral e altri.

Una volta configurato il modello, puoi iniziare a chattare con il tuo assistente direttamente dalla dashboard. Se vuoi parlargli tramite WhatsApp, Telegram, Discord o altri canali, anche quelli si configurano dalla dashboard.

## Perché è importante

Ci sono molte interfacce chat AI in giro. La maggior parte sono servizi cloud in cui i tuoi dati lasciano il tuo controllo nel momento in cui invii un messaggio. OpenClaw è diverso perché tutto resta sul tuo Appbox. L'assistente, la sua memoria, le tue conversazioni e i dati con cui lavora.

E poiché si trova accanto alle altre app con accesso al filesystem condiviso, può davvero fare cose in tutto il tuo stack self-hosted invece di limitarsi a rispondere a domande su di esso.

Siamo davvero curiosi di vedere cosa costruiranno le persone con questo. Se configuri un workflow interessante o trovi un modo creativo per collegare OpenClaw con le tue altre app, ci piacerebbe saperlo.
