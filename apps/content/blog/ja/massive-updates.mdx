# Appbox の新時代が始まります

何か月も裏側で作業し、夜遅くまで取り組み、コーヒーを飲みすぎた末に、ようやく私たちが作ってきたものをお見せできることを嬉しく思います。これは単なるアップデートではありません。Appbox を土台から作り直す取り組みです。

## 新しいデジタルホームへようこそ

今ご覧いただいているのは、まったく新しい Web サイトです！ついに古い WordPress 構成に別れを告げました。全体を、より高速で安全、そしてきちんとスケールする最新技術で再構築しています。

この新しいサイトは、きちんとしたドキュメント、定期的なブログ記事（この記事のようなもの）、そして最終的には新プラットフォームの frontend のホームになります。デザインは単にすっきりしただけではありません。実際に機能的で、必要な情報をずっと見つけやすくしています。

内部では Next.js と ContentLayer を使っており、非常に高速な性能と、ずっと良い開発体験を実現しています。このサイトは複数言語に対応し、SEO を自動処理し、XML と JSON の両方の RSS feeds を持ち、午前2時に真っ白な画面を見なくて済むよう dark mode も備えています。

## オープンソース化します！

私たちはオープンソースにも進みます！今読んでいるこの Web サイトのコードは公開されており、誰でも貢献できます。

これは私たちにとって大きな一歩です。私たちは常に透明性を大事にしてきましたが、今回は言葉だけでなく、コミュニティのためだけではなくコミュニティと一緒に作る姿勢を実際に示します。開発者の方で、自分が使うツールの形作りに関わりたいと思ったことがあるなら、今がその機会です。貢献ガイドラインや始めるために必要な情報は GitHub repo を確認してください。

## Appbox v2 の紹介

新しい Web サイトは始まりにすぎません。私たちは、これまでで最大のアップデートとなる Appbox v2 に取り組んでいます。

コンテナデプロイは常に私たちの中心でしたが、v2 では皆さんが求めてきた full VPS support によってさらに進化します。VPS apps は、制限のない完全な root access を提供します。環境を完全に制御したいなら、これこそ待っていたものです。

すでに1台のサーバーで動作しており、非常にうまく動いています。他のアプリと同じように、App Store から VPS apps をインストールできるようになります。これにより、Docker、Snap、Tailscale のような、現在は不可能なアプリもインストールできるようになります。より安全にし、性能を高めるために新しい仮想化機能を使っているため、当社サーバーを Debian 13 へ OS update する必要があります。この機能をリリースする前にその更新を展開する必要がありますが、近日中にさらに情報をお届けします。

これは、実績あるコンテナインフラと本物の VPS の柔軟性を組み合わせ、両方の良さを提供するものです。

## 独自ネットワークインフラの構築

本当に大きなニュースです。私たちは独自ネットワークを構築しています。

単にサーバーを増やしたり帯域をアップグレードしたりするだけではありません。独自の Autonomous System Number（[AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)）と独自 IP block（79.140.195.0/24）を取得し、完全に自分たちのネットワークを構築します。要するに、公式なインターネット上の identity と address space を持つ自分たちの ISP になることで、お客様のデータがインターネット上をどう流れるかを制御できるようになります [1]。

これは皆さんにとって何を意味するのでしょうか。私たちはすべてを制御し、Appbox ユーザー向けに最適化し、これまで不可能だった信頼性と性能を提供できます。

## AMS-IX とのピアリング

ネットワーク開発に合わせて、ネットワーク構成が整い次第、100G の接続で AMS-IX との peering を開始します。これにより、世界最大級のインターネット交換点へ直接アクセスできるようになり、全員にとって低 latency とより良い速度を意味します。主要インターネットプロバイダーへ裏道ではなく直通高速道路を持つようなものです [2]。

この種の接続性は通常、はるかに大きな企業にしか利用できませんが、私たちはどのプランでも enterprise-grade performance を提供したいと考えています。

ハードウェア購入の契約はまとまり、準備はできています。新しいハードウェアは遅くとも10月、早ければ9月に受け取る予定です。つまり、その後まもなく新ネットワークで稼働できる見込みです。

## 価格調整

これは本当に書きたくなかった部分です。このようなインフラを構築するには費用がかかり、その大部分は私たち自身で負担しますが、残念ながら新規・既存のお客様の両方に対して、価格を少し調整する必要があります。

この点は率直にお伝えしたいと思います。新しい価格はすでにサイト上で公開されており、既存のお客様には、これがどのように影響するかを近日中にご連絡します。

今は誰にとっても余裕がないことを分かっていますし、これは正直なところ私たちが下さなければならなかった最も難しい判断の1つでした。値上げは可能な限り小さく抑えています。この取り組みを成立させるために必要な最低限を除き、基本的には私たちが負担しています。

ただし、この価格変更にはそれ以上の価値を持たせます。性能と信頼性の違いを本当に感じられるはずです。さらに、これにより今後もネットワーク拡張と peering の追加を続けられるため、ここからさらに良くなっていきます。

新規のお客様向けには3か月前にこの更新価格を導入しており、反応は良好でした。そのため、手頃さを保ちつつ、皆さんにふさわしい改善へ資金を回すバランスが取れていると感じています。競争力も依然として高いです。たとえば人気の NG-4500 プランは月額 €2.08 の上昇にとどまり、競合の同等プランよりまだ大幅に安価です。

既存のお客様は、旧価格と新価格の差額のみをお支払いいただきます。プロモーション割引がある場合は、新しい基本価格の上にそのまま適用されます。特にこのような時期に継続してくださっていることに感謝しています。

割引構造は同じままです。
- 四半期払いで 5% off
- 6か月払いで 10% off
- 年払いで 15% off

自動支払いを設定している場合、メール受信後の次回請求サイクルで調整が行われます。お客様側で必要な作業はありません。

## さらに多くのストレージ！

ただ値上げするだけではありません。最小プランのストレージも 3TB から 3.5TB に増やします。16% の増加で、価格変更後も entry-level plan はさらにお得になります。

## 次に来るもの

これは Appbox にやってくるものの始まりにすぎません。今後数週間から数か月にわたり、新機能の展開、既存機能の改善、次世代アプリホスティングのためのインフラ構築を進めながら、定期的に更新をお届けします。

新しいネットワークインフラのテストは9月/10月に開始する予定で、進捗に応じて随時お知らせします。

ロードマップには次が含まれます。
- 制限のない VPS apps（Docker、Snap、Tailscale など）
- すべてを実際に網羅する、より良いドキュメント
- Appbox で人々が作っている面白いもののコミュニティショーケース
- インフラに関する定期的な技術ブログ記事
- 特定用途向けの新しいコンテナ化アプリ
- より良い監視・分析ツール
- すべてのプランにわたる強化されたセキュリティ機能

私たちと一緒に歩んでくれてありがとうございます。コミュニティなしには、これは何一つ実現できません。

さらなる更新を近日中にお届けします。
