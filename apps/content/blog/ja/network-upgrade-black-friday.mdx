# やりました - 私たちのネットワークが稼働しました！

4月に独自ネットワークインフラを構築していると発表したことを覚えていますか？昨日の時点で、ついに本番稼働したことをお知らせできてとても嬉しいです。

これは何か月もかけて準備してきたもので、正直なところ、すべてが形になるのを見るのは Appbox を始めて以来、最も満足感のある瞬間の1つでした。何が変わったのかを説明します。

## エンタープライズ級ハードウェア

当社インフラ全体に Nokia 7750 SR-1 routers を導入しました。Nokia のルーティング機器に詳しくない方のために説明すると、これらは世界最大級の ISP やデータセンターを支えるものと同じルーターです。大規模環境での信頼性と性能のために作られており、今ではお客様のすべてのトラフィックを処理しています。

これは一般消費者向けや prosumer 向けの機器ではありません。皆さんにふさわしい性能を提供するために必要な制御と能力を与えてくれる、本格的な enterprise networking hardware です。

## 独自 IP Space

現在、私たちは自社の Autonomous System（[AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)）配下の独自 IP アドレスでトラフィックをルーティングしています。つまり、お客様のデータがインターネット上をどのように移動するかを完全に制御できます。

なぜ重要なのでしょうか。Appbox ユーザーに特化して routing を最適化し、より良い DDoS protection を実装し、将来的には必要な方に専用 IP アドレスのような機能を提供できるからです。

## 100G AMS-IX Connectivity

大きなポイントです。現在、私たちは AMS-IX と毎秒100ギガビットで peering しています。AMS-IX は世界最大級のインターネット交換点の1つで、直接アクセスできることで、お客様のトラフィックは主要ネットワークやコンテンツプロバイダーへ可能な限り速い経路を取れます。

実際には、これは次を意味します。
- **低 latency** - 接続がより速く、応答性が高くなります
- **より良い速度** - 帯域が増えることで混雑が減ります
- **信頼性の向上** - direct peering により障害点が減ります

過去数週間、一部トラフィックでこれをテストしてきましたが、結果はまさに期待どおりでした。これからは全員がその恩恵を受けられます。

この節目を祝って（そして Black Friday weekend でもあるので）、これまでで最大のプロモーションを実施します。

より大きなプランへアップグレードしたいですか？こちらも対象です。コード **UPGRADE30** で **アップグレードが 30% off** になります。アップグレード料金は現在の請求サイクルに基づいて日割りされ、その後 30% 減額されます。アップグレード後の次の2か月も、通常価格から 30% off になります。

## VPS Apps - クリスマス前に登場予定

多くの方がこれを待っていることは分かっています。VPS apps は予定どおり進んでおり、クリスマス前に準備を整えることを目指しています。

以前の発表を見逃した方のために説明すると、VPS apps は制限のない完全な root access を提供します。Docker、Snap、Tailscale など、適切な system-level access が必要なものを実行できるようになります。VPS の柔軟性と、当社のワンクリックアプリデプロイを組み合わせたものです。

まだ発表できない別のものも準備中です。VPS リリースと一緒に、皆さんに気に入っていただけると思うサプライズ機能が来る、とだけ言っておきます。詳細は近日中にお知らせします。

## 次に来るもの

このネットワークアップグレードは土台にすぎません。インフラを適切に制御できるようになったことで、これまで不可能だった機能を構築し始められます。

- **Dedicated IP addresses** - 自分専用の IP が必要なユーザー向け
- **Advanced routing options** - トラフィックの流れをより細かく制御
- **Enhanced DDoS protection** - 新インフラを活用したより良いセキュリティ
- **Geographic optimization** - peering 関係の拡大に合わせた最適化

これらの機能が進展したら、随時お知らせします。
