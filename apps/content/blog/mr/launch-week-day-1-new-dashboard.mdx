## Custom Domains आता First Class आहेत

Custom domains आता install experience चा core भाग आहेत, awkward extra step नाहीत.

एखाद्या app ला domain लागत असल्यास, setup आता install dialog मध्येच clear guided flow सह होते:

1. तुमचा Appbox निवडा.
2. **Domain Type** निवडा:
   - तुमचा Appbox default domain वापरण्यासाठी **Appbox Domain**.
   - तुमच्या स्वतःच्या domains पैकी एक वापरण्यासाठी **Custom Domain**.
3. **Base Domain** निवडा.
4. वापरायचा **Subdomain** enter करा.
5. Custom domain निवडल्यास, Appbox add करायचा exact DNS record दाखवतो (host/name आणि target IP), आणि लगेच verify करू देतो.

Records guess करणे, docs copy करणे, आणि installs पुन्हा try करणे याऐवजी, app install करता त्याच ठिकाणी built-in DNS setup आणि verification step मिळतो.

## नवीन File Explorer (Scratch पासून built)

File explorer पूर्णपणे नव्याने विचार करून बनवला आहे. मागील experience मध्ये file management फक्त files delete करण्यापुरते limited होते. आता तो proper file manager आहे, ज्यात:

- uploads
- new folder creation
- rename
- delete
- download
- list/grid view switching
- breadcrumb navigation आणि contextual actions

म्हणून अतिरिक्त app install न करता तुमच्या appbox files natively manage करता येतात.

## संपूर्ण Dashboard मध्ये नवीन Design Language

Navigation, spacing, cards, typography, आणि interaction states मध्ये visual system refresh केले आहे. Target होते consistency आणि focus: कमी visual noise, stronger hierarchy, चांगली readability, आणि dark व light themes मध्ये अधिक modern product feel.
