# आम्ही केले - आमचे Network Live आहे!

April मध्ये आम्ही स्वतःचे network infrastructure build करत आहोत असे जाहीर केले होते, आठवते? कालपासून ते अखेर live झाले आहे हे सांगताना मला खूप आनंद होत आहे.

याला तयार व्हायला अनेक महिने लागले, आणि प्रामाणिकपणे सांगायचे तर सगळे एकत्र येताना पाहणे हे Appbox सुरू केल्यापासूनचे सर्वात satisfying moments पैकी एक होते. काय बदलले आहे ते सांगतो.

## Enterprise-Grade Hardware

आम्ही आमच्या infrastructure मध्ये Nokia 7750 SR-1 routers deploy केले आहेत. Nokia चे routing gear माहिती नसेल, तर हे जगातील काही सर्वात मोठ्या ISPs आणि data centers चालवणारे तेच routers आहेत. Scale वर reliability आणि performance साठी ते बनवलेले आहेत, आणि आता तुमचे सर्व traffic ते handle करत आहेत.

हे consumer किंवा prosumer equipment नाही. हे proper enterprise networking hardware आहे जे आम्हाला तुम्ही deserve करता तो performance देण्यासाठी आवश्यक control आणि capabilities देते.

## आमचा स्वतःचा IP Space

आता आम्ही आमच्या Autonomous System ([AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)) अंतर्गत आमच्या स्वतःच्या IP addresses मधून traffic route करत आहोत. याचा अर्थ तुमचा data internet वर कसा प्रवास करतो यावर आमचे पूर्ण control आहे.

हे महत्त्वाचे का? कारण आम्ही routing Appbox users साठी specifically optimize करू शकतो, चांगले DDoS protection implement करू शकतो, आणि भविष्यात ज्यांना गरज आहे त्यांच्यासाठी dedicated IP addresses सारखी features देऊ शकतो.

## 100G AMS-IX Connectivity

मोठी गोष्ट. आता आम्ही AMS-IX सोबत 100 gigabits per second वर peering करत आहोत. AMS-IX हे जगातील सर्वात मोठ्या internet exchanges पैकी एक आहे, आणि direct access असल्यामुळे तुमचे traffic major networks आणि content providers कडे शक्य तितक्या fastest route ने जाते.

प्रत्यक्षात याचा अर्थ:
- **कमी latency** - तुमचे connections faster आणि अधिक responsive
- **चांगले speeds** - अधिक bandwidth म्हणजे कमी congestion
- **सुधारित reliability** - direct peering म्हणजे fewer points of failure

गेल्या काही weeks मध्ये आम्ही traffic च्या subset सह हे test केले, आणि results नेमके अपेक्षेप्रमाणे आले. आता सगळ्यांना फायदा मिळतो.
