## Windows в Appbox

Когда мы запустили VPS в день 3 с пятью дистрибутивами Linux, естественный вопрос был: «а что насчет Windows?». Заставить Windows корректно работать как managed VPS в Appbox было серьезной технической задачей. Linux VM используют virtiofs для почти нативного file sharing между host и guest. Windows не поддерживает virtiofs. Поэтому мы построили интеграцию на базе Samba, которая создает authenticated SMB shares для каждой VM, давая Windows такой же бесшовный доступ к данным Appbox, как у Linux VPS, только через протокол, который Windows понимает нативно.

Результат — полноценная виртуальная машина Windows 11 Pro на KVM, где ваши данные Appbox подключены как диск `Z:` (`APPBOX_DATA`). Откройте File Explorer, перейдите в `Z:\`, и вы увидите папки `apps/` и `storage/` прямо там.

## Что это открывает

Windows VPS в Appbox открывает совершенно новую категорию сценариев:

- **ПО только для Windows** - запускайте приложения, у которых просто нет Linux-версий
- **.NET-разработка** - собирайте и тестируйте .NET-приложения в настоящей Windows-среде
- **Office и продуктивность** - запускайте Microsoft Office, Adobe tools или любое desktop software
- **Игровые серверы** - размещайте игровые серверы только для Windows
- **Remote desktop** - получайте доступ к полноценному Windows desktop из любого браузера через встроенную web console или подключайтесь через Microsoft Remote Desktop для нативного RDP-опыта
- **Тестирование и CI** - тестируйте свое ПО в Windows без поддержки собственного оборудования

Это полноценная установка Windows 11 Pro с administrator access. Если это работает в Windows, это работает здесь.

## Доступность по тарифам

У Windows 11 Pro довольно высокие требования к ресурсам, и она интенсивно использует disk I/O. Чтобы обеспечить хороший опыт для всех, Windows VPS сейчас доступна только на выделенных тарифах Appbox (NG-18000 и выше). В будущем мы можем расширить доступность на другие тарифы по мере оптимизации.

## Лицензирование

Windows 11 Pro VPS не включает product key. Windows будет работать неактивированной с постоянным watermark на рабочем столе, ограниченными настройками personalization и периодическими напоминаниями об activation, но в остальном полностью функциональна. Чтобы убрать эти ограничения, активируйте ее действительным ключом Windows 11 Pro. Мы рекомендуем приобрести его на [WinCDKey (~6 EUR)](https://wincdkey.com/product/microsoft-windows-11-professional-1-device-pc-software/).

## Работает с App Boost

Windows 11 Pro VPS полностью поддерживает **App Boost** из дня 2. Если вам нужна более высокая производительность для тяжелых workloads, поднимите уровень boost, чтобы выделить VM больше RAM и CPU.

## App Freeze

Сегодня выходит и вторая функция, которая относится ко всем приложениям, не только Windows.

<p align="center">
  <img src="/blog-images/day5-windows-detail-freeze.png" alt="Страница деталей Windows 11 Pro VPS с App Boost и новой кнопкой Freeze" width="560" className="mt-8 mb-10" />
</p>

**App Freeze** позволяет поставить любое установленное приложение на паузу и вернуть его app slots без потери данных. Приложение останавливается, его данные остаются на месте, а использованные слоты становятся доступными для других приложений.

Это решает проблему, которая постоянно всплывала: вы хотите попробовать новое приложение, но уже использовали все слоты. До сегодняшнего дня приходилось удалять что-то — и терять его данные — просто чтобы освободить достаточно слотов для эксперимента. Теперь можно заморозить приложение, использовать эти слоты для установки и тестирования чего-то нового, а затем разморозить исходное приложение, когда закончите. Все продолжится с того же места.

Кнопка Freeze находится прямо в разделе Actions на странице деталей любого приложения, рядом со Start, Stop и Restart. Заморозка останавливает приложение, помечает его как frozen и возвращает его слоты (включая boost slots) в доступный пул. Разморозка восстанавливает приложение и снова занимает слоты.

Несколько важных замечаний:

- Нельзя заморозить приложение, пока оно устанавливается, обновляется или удаляется
- Нельзя разморозить, если в Appbox недостаточно свободных слотов для покрытия стоимости
- Замороженные приложения сохраняют все данные и конфигурацию
