# OpenClaw finns nu på Appbox

Vi har precis lagt till något i appbutiken som jag personligen har varit peppad på ett tag.

[OpenClaw](https://openclaw.ai) är en open source-AI-assistent som körs helt på din egen infrastruktur. Tänk på den som ett privat, self-hosted alternativ till molnbaserade AI-assistenter, förutom att den här bor på din Appbox, minns dina konversationer och faktiskt kan göra saker åt dig.

Du pratar med den via de chattappar du redan använder. WhatsApp, Telegram, Discord, Slack, Signal, till och med iMessage. Den svarar med full kraft från modeller från Anthropic, OpenAI, Google, xAI, Mistral och andra. Du tar med din egen API-nyckel, väljer din modell, och allt stannar på din server.

Bara det vore användbart. Men anledningen till att jag skriver ett blogginlägg om den istället för att bara tyst lägga till den i butiken är vad som händer när du kombinerar den med resten av din Appbox.

## Den kan se dina andra appar

Här är vad som gör OpenClaw på Appbox annorlunda än att köra den någon annanstans.

Varje Appbox har ett delat datalager. Dina installerade appar får varsin katalog under en gemensam mount, och det finns ett delat lagringsområde som valfri app kan läsa från och skriva till. OpenClaw har åtkomst till samma delade filsystem. Det betyder att den kan se data från dina andra installerade appar och din allmänna lagring, inte via någon skör API-integration, utan via direkt filsystemåtkomst.

I praktiken är din AI-assistent inte isolerad i sin egen container utan kontext. Den sitter precis bredvid dina filer, databaser, medier, kod och konfigurationer.

## Den kan logga in i dina andra appar

Utöver det delade filsystemet har OpenClaw en fullständig headless browser (Chromium med Puppeteer) och alla vanliga databasklienter och CLI-verktyg inbyggda. Om du ger den inloggningsuppgifter till en av dina andra appar kan den faktiskt logga in och använda den åt dig.

Vill du uppdatera teamdokumentationen i Docmost? Säg till OpenClaw vad den ska skriva och ge den URL och inloggningsuppgifter. Den loggar in, navigerar till rätt sida och gör ändringarna. Behöver du en ny landing page på din Nginx & PHP-stack? Beskriv vad du vill ha så kan OpenClaw skriva HTML, CSS och JavaScript och sedan distribuera filerna direkt till din web root via det delade filsystemet. Kör du WordPress? Be den skriva och publicera ett blogginlägg, uppdatera inställningar för ett plugin eller göra om stilen på en sida. Den kan styra adminpanelen precis som du skulle göra.

Detta är inte heller begränsat till web UI:n. OpenClaw levereras med `mysql`, `psql`, `redis-cli`, `mongosh` och SSH-stöd, så den kan ansluta direkt till dina databaser och tjänster med de inloggningsuppgifter du tillhandahåller. Poängen är att OpenClaw inte behöver specialintegrationer eller plugins för varje app. Om du kan logga in i den kan din assistent också göra det.

## Hur det faktiskt ser ut

Låt mig gå igenom några mer specifika exempel på vad du kan göra när en AI-assistent har åtkomst till ditt Appbox-ekosystem.

### Underhåll av medieserver

Om du kör Plex, Jellyfin eller Emby levereras OpenClaw med `ffmpeg`, `mediainfo` och `exiftool` inbyggda. De är praktiska för att kontrollera filmetadata eller hitta kodningsproblem utan att behöva gräva igenom kataloger manuellt. Be den lista vad som finns i en mapp, kontrollera format eller hjälpa dig städa upp namngivningskonventioner så att din medieserver indexerar saker korrekt.

### Databasarbete utan smärtan

OpenClaws container kommer med `mysql`, `psql`, `sqlite3`, `redis-cli` och `mongosh` förinstallerade. Om du kör MariaDB kan du be OpenClaw fråga dina databaser, inspektera scheman, felsöka långsamma frågor eller hjälpa dig skriva migreringar. Allt via ett chattmeddelande på telefonen.

Kör du WordPress eller ett annat CMS med MariaDB bakom? Be OpenClaw kontrollera databasens hälsa, hitta föräldralösa postmetadata eller exportera specifikt innehåll. Du behöver inte SSH:a in någonstans eller minnas rätt CLI-flaggor. Beskriv bara vad du behöver på vanligt språk.

### GitLab och kodarbetsflöden

Om du har GitLab installerat kan OpenClaw arbeta direkt med dina repositories. Den har `git` inbyggt tillsammans med `ripgrep` och `fd` för snabb kodsökning. Be den söka igenom dina repos efter ett specifikt mönster, sammanfatta senaste commits eller hjälpa dig förstå obekant kod.

För team som använder GitLab för interna projekt är en AI-assistent som faktiskt kan läsa din kodbas och svara på frågor om den utan att skicka din kod till en tredjeparts molntjänst en meningsfull förbättring av utvecklingsarbetsflödet.

### Filorganisation och synk

Kör du Nextcloud, SFTPGo eller File Browser? OpenClaw kan hjälpa till att hantera filerna som flödar genom dessa tjänster. Den kan söka igenom dokument, organisera om katalogstrukturer, hitta dubbletter och flytta saker mellan din delade lagring och appspecifika kataloger.

Den har också `rclone` tillgängligt, så om du behöver sätta upp synkjobb eller flytta data mellan molnlagringsleverantörer kan du beskriva vad du vill och låta OpenClaw hantera konfigurationen.

### Övervakning och diagnostik

Om du har installerat Uptime Kuma för att övervaka dina tjänster ger OpenClaw dig ett konversationslager ovanpå din övervakning. Utöver att bara ta emot larm kan du be OpenClaw köra nätverksdiagnostik. Den levereras med `ping`, `dig`, `nslookup`, `traceroute`, `mtr` och till och med `nmap` och `tcpdump` för djupare undersökningar.

Går något ner? Skicka ett meddelande till din assistent, be den kontrollera vad som händer och få en förklaring på vanligt språk istället för att stirra på dashboards.

### Musik- och ljudboksbibliotek

Navidrome- och audiobookshelf-användare kan dra nytta av OpenClaws medieverktyg. Be den skanna ditt musikbibliotek efter filer med saknade eller felaktiga metadatataggar, identifiera spår som kan behöva omkodning eller hjälpa till att organisera ljudbokskapitel som inte delades upp korrekt. Kombinationen `ffmpeg` och `exiftool` hanterar det mesta du behöver.

### Jupyter och dataarbete

Om du använder Jupyter Notebook för dataanalys kompletterar OpenClaw det fint. Den har Python 3 med pip- och venv-stöd, så den kan hjälpa dig skriva skript, bearbeta datafiler i din delade lagring och till och med köra snabba analyser direkt. Tänk på den som ett konversationsgränssnitt till samma data som dina notebooks arbetar med.

### Infrastructure as Code

För de mer tekniskt äventyrliga levereras OpenClaw med `docker`, `kubectl`, `terraform` och `ansible` CLI:er. Om du kör en VPS på Appbox och hanterar infrastruktur kan du använda OpenClaw som en konversationsbaserad ops-assistent. Beskriv vad du vill distribuera och låt den hjälpa till att generera och tillämpa konfigurationer.

## Persistent minne

En av sakerna som skiljer OpenClaw från att bara ha ett chattfönster till en AI-modell är att den minns dig. Den bygger upp kontext över tid: dina preferenser, hur du vill att saker ska organiseras, projekten du arbetar med och de namngivningskonventioner du använder. Ju mer du interagerar med den, desto mer användbar blir den.

Det här är särskilt kraftfullt på Appbox eftersom kontexten inte bara är konversation. Den är grundad i din faktiska miljö. Dina filer, dina appar, dina data.

## Förinstallerade färdigheter

OpenClaw har ett skill-system med 50+ communitybyggda tillägg tillgängliga på [ClawHub](https://clawhub.ai/skills). Skills ger agenten specialiserade förmågor för specifika appar, verktyg eller arbetsflöden.

På Appbox ger vi dig inte bara den nakna agenten. Imagen levereras med en kurerad uppsättning färdigheter redan installerade så att OpenClaw förstår din miljö direkt:

- **Plattformsmedvetenhet**: känner till Appbox delade datalayout, tillgängliga CLI-verktyg och hur den guidar dig till support eller VPS-uppgraderingar vid behov
- **Appspecifika färdigheter**: förinstallerade för WordPress, Nginx, GitLab, Nextcloud, Emby, MariaDB, Tautulli, Mautic, WireGuard och Rocket.Chat, så agenten redan vet hur den arbetar med dessa appar när du har dem installerade
- **SSH-grunder**: för att ansluta till dina VPS:er eller fjärrservrar direkt från agenten
- **Webbläsarautomation (Playwright MCP)**: driver möjligheten att logga in i och styra dina andra appar via deras web UI:n
- **Webbsökning och dokumentationsuppslag**: sökning med flera motorer och ClawDDocs för att hämta referensmaterial i stunden
- **Självförbättring**: färdigheter för att upptäcka nya skills på ClawHub och granska dem före installation

Du kan installera fler skills när som helst. Hitta en på [ClawHub](https://clawhub.ai/skills), kopiera länken och klistra in den till din agent. Den laddar ner och installerar färdigheten själv.

## Kom igång

Installera OpenClaw från Appbox app store som vilken annan app som helst. Den tar två slots.

Efter installationen ser du en Gateway Token på appens detaljsida. Det är din dashboard-inloggning. Öppna dashboard-URL:en, ange token och du är inne.

Därifrån är det enda du behöver konfigurera din AI-modellleverantör. Lägg till en API-nyckel från den leverantör du föredrar. OpenRouter är den enklaste startpunkten eftersom en nyckel ger dig åtkomst till många modeller. Vi har skrivit en [full setup guide](/docs/apps/openclaw-model-provider-setup) som går igenom hur du skaffar API-nycklar från OpenRouter, OpenAI, Anthropic, Google, xAI, Mistral och andra.

När din modell är konfigurerad kan du börja chatta med din assistent direkt från dashboarden. Om du vill prata med den via WhatsApp, Telegram, Discord eller andra kanaler konfigureras de också via dashboarden.

## Varför detta spelar roll

Det finns gott om AI-chattgränssnitt där ute. De flesta är molntjänster där dina data lämnar din kontroll så snart du skickar ett meddelande. OpenClaw är annorlunda eftersom allt stannar på din Appbox. Assistenten, dess minne, dina konversationer och de data den arbetar med.

Och eftersom den ligger bredvid dina andra appar med delad filsystemåtkomst kan den faktiskt göra saker över hela din self-hosted stack istället för att bara svara på frågor om den.

Vi är verkligen nyfikna på att se vad folk bygger med detta. Om du sätter upp ett intressant arbetsflöde eller hittar ett kreativt sätt att koppla OpenClaw till dina andra appar vill vi gärna höra om det.
