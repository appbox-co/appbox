# Launch Week రోజు 1: కొత్త Dashboard Live

Launch Week సోమవారం ప్రారంభమవుతుందని మేము చెప్పాము. Day 1 ఇప్పుడు live.

## Custom Domains ఇప్పుడు First Class

Custom domains ఇప్పుడు install experience లో core భాగం, awkward extra step కాదు.

ఒక app కు domain అవసరమైతే, setup ఇప్పుడు install dialog లోనే clear guided flow తో జరుగుతుంది:

1. మీ Appbox ఎంచుకోండి.
2. **Domain Type** ఎంచుకోండి:
   - మీ Appbox default domain ఉపయోగించడానికి **Appbox Domain**.
   - మీ స్వంత domains లో ఒకటి ఉపయోగించడానికి **Custom Domain**.
3. **Base Domain** ఎంచుకోండి.
4. ఉపయోగించాలనుకునే **Subdomain** enter చేయండి.
5. custom domain ఎంచుకుంటే, add చేయాల్సిన ఖచ్చితమైన DNS record (host/name మరియు target IP) Appbox చూపిస్తుంది, తర్వాత వెంటనే verify చేయడానికి అనుమతిస్తుంది.

records ను ఊహించడం, docs copy చేయడం, installs retry చేయడం బదులుగా, app install చేసే అదే చోట built-in DNS setup మరియు verification step లభిస్తుంది.

## కొత్త File Explorer (మొదటి నుండి నిర్మించబడింది)

file explorer మొదటి నుంచీ మళ్లీ ఆలోచించి నిర్మించబడింది. పాత experience లో file management కేవలం files delete చేయడానికే పరిమితం. ఇప్పుడు ఇది proper file manager, ఇందులో:

- uploads
- new folder creation
- rename
- delete
- download
- list/grid view switching
- breadcrumb navigation మరియు contextual actions

అదనపు app install చేయకుండా మీ appbox files ను native గా నిజంగా manage చేయవచ్చు.

## Dashboard అంతటా కొత్త Design Language

navigation, spacing, cards, typography, మరియు interaction states అంతటా visual system refresh చేయబడింది. లక్ష్యం consistency మరియు focus: తక్కువ visual noise, బలమైన hierarchy, మెరుగైన readability, dark మరియు light themes రెండింటిలో మరింత modern product feel.

ఇకపై అర్ధరాత్రి apps install చేస్తూ కళ్లకు గుచ్చే తెల్ల screen ఉండదు!

## మెరుగైన Daily UX: Theme, Language, మరియు వేగవంతమైన Shortcuts

ఈ release ప్రతి రోజు dashboard ఉపయోగించడం మరింత సౌకర్యంగా చేసే quality-of-life features కూడా కలిగి ఉంది:

- **Native language switching** నేరుగా UI లో (కాలక్రమంలో మరిన్ని languages జోడించబడతాయి).
