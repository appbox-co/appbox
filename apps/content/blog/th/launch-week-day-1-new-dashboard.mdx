# Launch Week Day 1: Dashboard ใหม่พร้อมใช้งานแล้ว

เราบอกว่า Launch Week จะเริ่มวันจันทร์ และตอนนี้ Day 1 ก็เปิดตัวแล้ว

วันนี้เรากำลังปล่อยประสบการณ์ dashboard ใหม่ของ Appbox นี่คือการอัปเกรด UI และ workflow ครั้งใหญ่ทั่ว control panel โดยมีเป้าหมายง่าย ๆ เพียงอย่างเดียว: ทำให้การใช้งาน appbox ของคุณในแต่ละวันเร็วขึ้น ชัดเจนขึ้น และน่ารำคาญน้อยลง

การอัปเดตนี้พัฒนามานานมาก และผมดีใจมากที่ในที่สุดเราก็ส่งมันถึงมือคุณได้

## ศูนย์ควบคุมใหม่

ก่อนวันนี้ ข้อมูลเหล่านี้กระจายอยู่ตามหน้าและ flow แยกกัน วันนี้เราขอแนะนำ dashboard เฉพาะที่รวมทุกอย่างไว้ในมุมมองการทำงานเดียว:

- blog และการอัปเดต platform ล่าสุด
- การ์ดภาพรวม service ของคุณ
- notification ของคุณ
- app ใหม่ล่าสุดและ app ที่เพิ่งอัปเดต

ดังนั้นแทนที่จะต้องสลับไปมาระหว่าง tab คุณสามารถเปิดหน้าเดียวและเห็นสิ่งสำคัญได้ทันที

## Custom Domains เป็นส่วนหลักแล้ว

Custom domains ตอนนี้เป็นส่วนหลักของประสบการณ์ install ไม่ใช่ขั้นตอนเสริมที่น่าอึดอัด

เมื่อ app ต้องใช้ domain การตั้งค่าจะเกิดขึ้นโดยตรงใน install dialog พร้อม flow แนะนำที่ชัดเจน:

1. เลือก Appbox ของคุณ
2. เลือก **Domain Type**:
   - **Appbox Domain** เพื่อใช้ domain เริ่มต้นของ Appbox ของคุณ
   - **Custom Domain** เพื่อใช้ domain ของคุณเอง
3. เลือก **Base Domain**
4. ใส่ **Subdomain** ที่คุณต้องการใช้
5. หากคุณเลือก custom domain, Appbox จะแสดง DNS record ที่ต้องเพิ่มอย่างชัดเจน (host/name และ target IP) แล้วให้คุณ verify ได้ทันที

แทนที่จะต้องเดา record เอง คัดลอก docs และ retry การ install คุณจะได้ขั้นตอน setup และ verification ของ DNS ที่ built-in อยู่ในที่เดียวกับที่คุณ install app

## File Explorer ใหม่ (สร้างใหม่ตั้งแต่ต้น)

file explorer ถูกคิดใหม่และสร้างใหม่ตั้งแต่ต้น ในประสบการณ์ก่อนหน้า การจัดการไฟล์จำกัดอยู่แค่การลบไฟล์ ตอนนี้มันเป็น file manager ที่เหมาะสมพร้อม:

- uploads
- การสร้าง folder ใหม่
- rename
- delete
- download
- การสลับมุมมอง list/grid
- breadcrumb navigation และ contextual actions

ดังนั้นคุณจึงสามารถจัดการไฟล์ appbox ของคุณแบบ native ได้จริง ๆ แทนที่จะต้องติดตั้ง app เพิ่ม

## การยืนยันตัวตนแบบ Two-Factor

ความปลอดภัยไม่ควรเป็นตัวเลือกในปี 2026 ตั้งแต่วันนี้ account Appbox รองรับ two-factor authentication แบบ built-in

คุณสามารถเปิดใช้และจัดการ 2FA ได้โดยตรงจากพื้นที่ account ของคุณ รวมถึงการจัดการ recovery code สำหรับการกู้คืน account

## Design Language ใหม่ทั่ว Dashboard

ระบบ visual ได้รับการ refresh ทั่ว navigation, spacing, cards, typography และ interaction states เป้าหมายคือความสม่ำเสมอและ focus: visual noise น้อยลง hierarchy ชัดขึ้น อ่านง่ายขึ้น และให้ความรู้สึกเป็น product สมัยใหม่มากขึ้นทั้งใน dark และ light themes

ต่อไปคุณจะไม่ต้องเจอหน้าจอขาวแสบตาตอน install app ตอนเที่ยงคืนอีกแล้ว!
