# ยุคใหม่เริ่มต้นที่ Appbox

หลังจากหลายเดือนของการทำงานเบื้องหลัง นอนดึกหลายคืน และดื่มกาแฟมากเกินไป ผมตื่นเต้นที่จะได้แสดงให้คุณเห็นในที่สุดว่าเรากำลังสร้างอะไรอยู่ นี่ไม่ใช่แค่อีก update หนึ่ง แต่นี่คือการสร้าง Appbox ใหม่ตั้งแต่รากฐาน

## ยินดีต้อนรับสู่บ้านดิจิทัลใหม่ของเรา

คุณกำลังดู website ใหม่เอี่ยมของเรา! ในที่สุดเราก็บอกลา setup WordPress เก่าของเราแล้ว ทั้งหมดถูกสร้างใหม่ด้วยเทคโนโลยีสมัยใหม่ที่เร็วกว่า ปลอดภัยกว่า และ scale ได้จริง

site ใหม่นี้จะกลายเป็นบ้านสำหรับ documentation ที่เหมาะสม, blog posts เป็นประจำ (เหมือนบทความนี้) และในที่สุดคือ frontend สำหรับ platform ใหม่ของเรา design ไม่ใช่แค่สะอาดขึ้น แต่ใช้งานได้จริงและทำให้หาเรื่องต่าง ๆ ง่ายขึ้นมาก

เราใช้ Next.js พร้อม ContentLayer อยู่เบื้องหลัง ซึ่งให้ performance ที่เร็วมากและ development experience ที่ดีกว่ามาก site รองรับหลายภาษา จัดการ SEO ให้อัตโนมัติ มี RSS feeds ทั้ง XML และ JSON และรวม dark mode เพราะไม่มีใครควรต้องรับมือกับหน้าจอขาวจ้าในเวลาตีสอง

## เรากำลัง Open Source!

เรายังกำลัง open source ด้วย! website ที่คุณกำลังอ่านอยู่ตอนนี้? code เป็น public และทุกคนสามารถ contribute ได้

นี่เป็นเรื่องใหญ่สำหรับเรา เราพยายามโปร่งใสมาโดยตลอด แต่นี่คือการลงมือทำจริงตามที่พูด และสร้างร่วมกับ community แทนที่จะสร้างเพื่อพวกเขาอย่างเดียว หากคุณเป็น developer และเคยอยากช่วยกำหนดทิศทางเครื่องมือที่คุณใช้ ตอนนี้คุณทำได้ ดู GitHub repo ของเราเพื่ออ่าน contribution guidelines และทุกอย่างที่ต้องใช้ในการเริ่มต้น

## ขอแนะนำ Appbox v2

website ใหม่เป็นเพียงจุดเริ่มต้น เรากำลังทำ Appbox v2 ซึ่งเป็น update ที่ใหญ่ที่สุดที่เราเคยทำ

Container deployment เป็นสิ่งที่เราถนัดมาตลอด แต่ v2 ไปไกลกว่านั้นด้วยสิ่งที่ทุกคนขอมา: full VPS support VPS apps จะให้ root access แบบสมบูรณ์โดยไม่มีข้อจำกัด หากคุณต้องการควบคุม environment ของคุณทั้งหมด นี่คือสิ่งที่คุณรออยู่

เราได้ทำให้มันทำงานบน server หนึ่งแล้วและมันทำงานได้ดีมาก คุณจะสามารถ install VPS apps ได้เหมือน app อื่น ๆ จาก app store สิ่งนี้จะให้คุณ install app ที่ปัจจุบันยังทำไม่ได้ เช่น Docker, Snap และ Tailscale มันต้องการ OS update บน servers ของเราเป็น Debian 13 เพราะเราใช้ virtualization features ใหม่บางอย่างเพื่อให้ปลอดภัยขึ้นและเพิ่ม performance เราจะต้อง rollout update นั้นก่อนเปิดตัว feature แต่จะมี update เพิ่มเติมเกี่ยวกับเรื่องนี้เร็ว ๆ นี้

สิ่งนี้รวม container infrastructure ที่พิสูจน์แล้วของเราเข้ากับความยืดหยุ่นแบบ VPS จริง ให้คุณได้สิ่งที่ดีที่สุดจากทั้งสองโลก

## สร้าง Network Infrastructure ของเราเอง

นี่คือข่าวใหญ่จริง ๆ เรากำลังสร้าง network ของเราเอง

ไม่ใช่แค่เพิ่ม servers หรืออัปเกรด bandwidth เรามี Autonomous System Number ของเราเอง ([AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)) และ IP block ของเราเอง (79.140.195.0/24) เพื่อสร้าง network ที่เป็นของเราอย่างสมบูรณ์ โดยพื้นฐานแล้วเรากำลังกลายเป็น ISP ของเราเอง พร้อม official internet identity และ address space ของเรา เพื่อให้เราควบคุมได้ว่าข้อมูลของคุณเดินทางผ่าน internet อย่างไร [1]

สิ่งนี้หมายความว่าอย่างไรสำหรับคุณ? เราควบคุมทุกอย่างได้ optimize เฉพาะสำหรับผู้ใช้ Appbox และส่งมอบ reliability กับ performance ที่ก่อนหน้านี้เป็นไปไม่ได้

## การ Peering กับ AMS-IX

เพื่อให้สอดคล้องกับการพัฒนา network ของเรา เมื่อเราตั้งค่า network เสร็จ เราจะเริ่ม peering กับ AMS-IX สำหรับ connectivity 100G สิ่งนี้ให้เราเข้าถึง internet exchange ที่ใหญ่ที่สุดแห่งหนึ่งของโลกโดยตรง ซึ่งหมายถึง latency ต่ำลงและ speed ดีขึ้นสำหรับทุกคน ลองคิดเหมือนมีทางด่วนตรงไปยัง internet providers รายใหญ่ แทนที่จะต้องใช้ถนนหลังบ้าน [2]

connectivity แบบนี้โดยปกติจะมีให้เฉพาะบริษัทขนาดใหญ่กว่ามาก แต่เราต้องการมอบ performance ระดับ enterprise ไม่ว่าคุณจะอยู่ plan ใด

ตอนนี้เราจัดการ agreement สำหรับการซื้อ hardware ของเราเรียบร้อยแล้ว ดังนั้นเราพร้อมเดินหน้า เราจะได้รับ hardware ใหม่ภายใน October อย่างช้าที่สุด หรืออาจเร็วที่สุดใน September นี่หมายความว่าเราควรเริ่มรันบน network ใหม่ได้ไม่นานหลังจากนั้น

## การปรับราคา

นี่คือส่วนที่เราไม่อยากเขียนจริง ๆ การสร้าง infrastructure แบบนี้มีค่าใช้จ่ายสูง และแม้ว่าเราจะรับภาระต้นทุนส่วนใหญ่เอง เราจำเป็นต้องปรับราคาเล็กน้อยสำหรับทั้งลูกค้าใหม่และลูกค้าเดิม

ผมอยากพูดตรง ๆ เรื่องนี้ ราคาใหม่ live อยู่บน site แล้ว และเราจะติดต่อลูกค้าเดิมเร็ว ๆ นี้พร้อมรายละเอียดว่าสิ่งนี้กระทบพวกเขาอย่างไร

เรารู้ว่าตอนนี้ทุกคนมีภาระหนัก และนี่เป็นหนึ่งในการตัดสินใจที่ยากที่สุดที่เราต้องทำจริง ๆ เราพยายามให้การเพิ่มราคาน้อยที่สุดเท่าที่ทำได้ โดยพื้นฐานแล้วเราครอบคลุมทุกอย่างยกเว้นขั้นต่ำสุดที่ต้องมีเพื่อให้สิ่งนี้เกิดขึ้นได้

แต่เราจะทำให้การขึ้นราคานี้คุ้มค่าเกินกว่านั้น คุณจะสังเกตเห็นความแตกต่างด้าน performance และ reliability อย่างชัดเจน อีกทั้งสิ่งนี้หมายความว่าเราจะสามารถขยาย network และเพิ่ม peering ต่อไปได้ ดังนั้นสิ่งต่าง ๆ จะดีขึ้นเรื่อย ๆ จากนี้

เราเปิดใช้ราคาใหม่เหล่านี้สำหรับลูกค้าใหม่เมื่อ 3 เดือนก่อน และ feedback ออกมาดี ดังนั้นเรารู้สึกว่าเราได้ balance ที่ดีระหว่างการคงราคาให้เข้าถึงได้และการ funding improvements ที่คุณสมควรได้รับ เรายังคงแข่งขันได้มาก ตัวอย่างเช่น plan NG-4500 ยอดนิยมของเราเพิ่มขึ้นเพียง €2.08 ต่อเดือน ซึ่งยังน้อยกว่ามากเมื่อเทียบกับ plan คล้ายกันจากคู่แข่ง

สำหรับลูกค้าเดิม คุณจะจ่ายเฉพาะส่วนต่างระหว่างราคาเก่าและราคาใหม่ หากคุณมี promotional discounts ส่วนลดเหล่านั้นยังอยู่บน base price ใหม่ เราขอบคุณใน loyalty ของคุณ โดยเฉพาะในช่วงเวลาแบบนี้

โครงสร้างส่วนลดของเรายังคงเหมือนเดิม:
- ลด 5% สำหรับการชำระเงินรายไตรมาส
- ลด 10% สำหรับการชำระเงิน 6 เดือน
- ลด 15% สำหรับการชำระเงินรายปี

หากคุณตั้งค่า automatic payments ไว้ การปรับจะเกิดขึ้นกับ billing cycle ถัดไปหลังจากที่คุณได้รับ email คุณไม่ต้องทำอะไร

## Storage เพิ่มขึ้น!

แต่ไม่ใช่ทุกอย่างจะเป็นเรื่องจ่ายเพิ่ม เรายังเพิ่ม storage ใน plan ต่ำสุดของเราจาก 3TB เป็น 3.5TB นั่นคือเพิ่มขึ้น 16% ซึ่งทำให้ entry-level plan ของเราคุ้มค่าขึ้นอีกแม้มีการเปลี่ยนราคา

## ต่อไปคืออะไร?

นี่เป็นเพียงจุดเริ่มต้นของสิ่งที่กำลังมาถึง Appbox ในอีกไม่กี่สัปดาห์และเดือนข้างหน้า คุณจะเห็น update เป็นประจำเมื่อเรา rollout features ใหม่ ปรับปรุงสิ่งที่มีอยู่ และสร้าง infrastructure สำหรับยุคถัดไปของ app hosting ต่อไป

เราวางแผนจะเริ่มทดสอบ network infrastructure ใหม่ของเราใน September/October และเราจะอัปเดตให้คุณทราบเมื่อมีความคืบหน้า

roadmap ของเรารวมถึง:
- VPS apps ที่จะไม่มีข้อจำกัด (เช่น Docker, Snap และ Tailscale)
- documentation ที่ดีขึ้นและครอบคลุมทุกอย่างจริง ๆ
- community showcase สำหรับสิ่งเจ๋ง ๆ ที่ผู้คนสร้างด้วย Appbox
- technical blog posts เป็นประจำเกี่ยวกับ infrastructure ของเรา
- containerized apps ใหม่สำหรับ use case เฉพาะ
- monitoring และ analytics tools ที่ดีขึ้น
- security features ที่ยกระดับขึ้นในทุก plans

ขอบคุณที่อยู่กับเรา หากไม่มี community ของเรา สิ่งนี้ทั้งหมดคงเป็นไปไม่ได้
