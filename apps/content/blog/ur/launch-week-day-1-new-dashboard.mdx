## Custom Domains اب First Class ہیں

Custom domains اب install experience کا core حصہ ہیں، کوئی awkward extra step نہیں۔

جب app کو domain require ہو، setup اب clear guided flow کے ساتھ install dialog میں directly ہوتا ہے:

1. اپنا Appbox select کریں۔
2. **Domain Type** choose کریں:
   - اپنے Appbox default domain کے لیے **Appbox Domain**۔
   - اپنے domains میں سے کسی کے لیے **Custom Domain**۔
3. **Base Domain** pick کریں۔
4. desired **Subdomain** enter کریں۔
5. اگر آپ نے custom domain select کیا ہے، Appbox exact DNS record دکھاتا ہے جو add کرنا ہے (host/name اور target IP)، پھر اسے immediately verify کرنے دیتا ہے۔

Records manually guess کرنے، docs copy کرنے، اور installs retry کرنے کے بجائے، آپ کو app install کرنے والی same جگہ پر built-in DNS setup اور verification step ملتا ہے۔
