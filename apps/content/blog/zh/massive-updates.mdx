# Appbox 的新时代开始了

经过数月的幕后工作、熬夜和喝下太多咖啡之后，我终于很兴奋地向你展示我们一直在构建的东西。这不只是又一次更新。这是我们从地基开始重建 Appbox。

## 欢迎来到我们的新数字家园

你现在看到的是我们的全新网站！我们终于告别了旧的 WordPress 设置。整个网站都用更快、更安全、也真正能扩展的现代技术重建。

这个新站点将成为正式文档、定期博客文章（就像这篇），并最终成为新平台前端的所在地。设计不只是更清爽，而是真的更实用，也让查找内容容易得多。

底层我们使用 Next.js 和 ContentLayer，这带来了非常快的性能和更好的开发体验。站点支持多语言，自动处理 SEO，同时提供 XML 和 JSON 格式 RSS feed，并包含深色模式，因为没人应该在凌晨 2 点还要面对刺眼白屏。

## 我们要开源了！

我们也要开源了！你正在阅读的这个网站？代码是公开的，任何人都可以贡献。

这对我们来说意义重大。我们一直努力保持透明，但这次是真正把话落到实处：不是只为社区构建，而是与社区一起构建。如果你是开发者，并且一直想帮助塑造自己使用的工具，现在可以了。查看我们的 GitHub 仓库，里面有贡献指南和开始所需的一切。

## 介绍 Appbox v2

新网站只是开始。我们正在开发 Appbox v2，这是我们做过的最大更新。

容器部署一直是我们的核心，但 v2 会进一步实现大家一直要求的完整 VPS 支持。VPS apps 将提供没有限制的完整 root 访问权限。如果你需要完全控制自己的环境，这就是你一直在等的东西。

我们已经在一台服务器上跑通，而且效果很好。你将能够像安装 app store 里的其他应用一样安装 VPS apps。这会让你安装目前不可能安装的应用，例如 Docker、Snap 和 Tailscale。由于我们使用了一些新的虚拟化功能来提高安全性和性能，这需要将服务器 OS 更新到 Debian 13。我们需要先完成该更新才能发布功能，但很快会有更多进展。

这会把我们成熟的容器基础设施与真正的 VPS 灵活性结合起来，让你同时获得两者优势。

## 构建我们自己的网络基础设施

真正的大消息是，我们正在建设自己的网络。

不是只添加更多服务器或升级带宽。我们已经拥有自己的 Autonomous System Number（[AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)）和自己的 IP block（79.140.195.0/24），用于构建完全属于我们的网络。基本上，我们正在成为自己的 ISP，拥有官方互联网身份和地址空间，因此可以控制你的数据如何穿越互联网 [1]。

这对你意味着什么？我们可以控制一切，专门为 Appbox 用户优化，并提供以前不可能实现的可靠性和性能。

## 与 AMS-IX Peering

配合我们的网络建设，在网络设置完成后，我们会开始与 AMS-IX 进行 100G 连接的 peering。这让我们能直接接入全球最大的互联网交换中心之一，也意味着所有用户都能获得更低延迟和更好速度。可以把它想象成通往主要互联网提供商的直达高速公路，而不是走小路 [2]。

这种连接通常只有大得多的公司才能获得，但无论你使用哪个套餐，我们都希望提供企业级性能。

我们已经完成硬件采购协议，所以准备好了。我们最迟会在 10 月收到新硬件，也可能早到 9 月。这意味着不久之后我们应该就能在新网络上运行。

## 价格调整

这是我们真的不想写的部分。构建这样的基础设施成本很高，虽然大部分成本由我们自己承担，但很遗憾，我们需要对新客户和现有客户的价格做一次小幅调整。

我想坦诚说明。新价格已经在网站上线，我们很快会联系现有客户，详细说明这会如何影响他们。

我们知道现在每个人都不轻松，而这确实是我们做过最艰难的决定之一。我们已经尽可能把涨幅控制得很小，基本上只覆盖让这件事能运转起来所需的最低部分。

不过，我们会让这次涨价物有所值。你会真正感受到性能和可靠性的差异。而且这意味着我们可以继续扩展网络并增加更多 peering，所以从这里开始只会越来越好。

我们 3 个月前已经面向新客户启用了这些更新后的价格，反馈不错，所以我们认为现在在保持可负担和为你应得的改进提供资金之间取得了不错平衡，而且仍然非常有竞争力。例如，我们热门的 NG-4500 套餐每月只上涨 €2.08，仍然远低于竞争对手的可比套餐。

对于现有客户，你只需支付旧价和新价之间的差额。如果你有任何促销折扣，它们会继续叠加在新的基础价格上。尤其在这样的时期，我们非常感谢你的忠诚。

我们的折扣结构保持不变：
- 季付 5% 折扣
- 6 个月付 10% 折扣
- 年付 15% 折扣

如果你设置了自动付款，收到邮件后，下一个计费周期会自动应用调整。你无需做任何事。

## 更多存储！

不过并不全是多付钱。我们也会把最低套餐的存储从 3TB 提升到 3.5TB。增长 16%，即使价格变化，也让入门套餐变得更划算。

## 接下来是什么？

这只是 Appbox 即将到来的开始。在接下来的几周和几个月，你会看到我们定期发布更新，推出新功能、改进现有功能，并继续构建下一代应用托管所需的基础设施。

我们计划在 9 月/10 月开始测试新的网络基础设施，并会随着进展持续更新。

我们的路线图包括：
- 不受限制的 VPS apps（例如 Docker、Snap 和 Tailscale）
- 真正覆盖所有内容的更好文档
- 社区展示，展示大家用 Appbox 构建的有趣东西
- 关于我们基础设施的定期技术博客文章
- 面向特定使用场景的新容器化应用
- 更好的监控和分析工具
- 覆盖所有套餐的增强安全功能

感谢你一直支持我们。没有我们的社区，这一切都不可能实现。

更多更新很快到来。
