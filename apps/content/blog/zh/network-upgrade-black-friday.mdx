# 我们做到了 - 新网络已上线！

还记得 4 月我们宣布正在建设自己的网络基础设施吗？现在我很高兴地告诉你，截至昨天，它终于正式上线了。

这件事已经筹备了好几个月。说实话，看着它全部拼起来，是我们创办 Appbox 以来最令人满足的时刻之一。让我来讲讲发生了哪些变化。

## 企业级硬件

我们已经在基础设施中部署了 Nokia 7750 SR-1 路由器。如果你不熟悉 Nokia 的路由设备，它们是支撑全球一些最大 ISP 和数据中心的同类路由器。它们为大规模可靠性和性能而构建，现在正在处理你的全部流量。

这不是消费级或 prosumer 设备。这是真正的企业级网络硬件，给了我们交付你应得性能所需的控制力和能力。

## 我们自己的 IP 空间

现在我们通过自己的 Autonomous System（[AS209870](https://apps.db.ripe.net/db-web-ui/lookup?source=RIPE&type=aut-num&key=AS209870)）下的自有 IP 地址路由流量。这意味着我们可以完全控制你的数据如何穿越互联网。

为什么这很重要？因为我们可以专门为 Appbox 用户优化路由，实现更好的 DDoS 防护，并在未来为有需要的用户提供 dedicated IP addresses 等功能。

## 100G AMS-IX 连接

重头戏。我们现在以每秒 100 gigabits 的容量与 AMS-IX peering。AMS-IX 是全球最大的互联网交换中心之一，直接接入意味着你的流量可以通过最快路径到达主要网络和内容提供商。

具体来说，这意味着：
- **更低延迟** - 连接更快、响应更灵敏
- **更好速度** - 更多带宽意味着更少拥塞
- **更高可靠性** - 直接 peering 意味着故障点更少

过去几周我们已经用一部分流量测试过它，结果正是我们所希望的。现在所有人都能受益。
