يستخدم `opendocs` مكتبة [`rehype-pretty-code`](https://github.com/atomiks/rehype-pretty-code)
للتعامل مع الكود داخل ملفات MDX. يتيح لك ذلك تخصيص طريقة عرض الكود بدرجة كبيرة!

لمزيد من المعلومات، راجع الوثائق الرسمية والأمثلة!

## العناوين

**مثال:**

```js title="apps/web/src/index.tsx"
import { App } from './App';
```

**الكود:**

````mdx
```js title="apps/web/src/index.tsx"
import { App } from './App';
```
````

## تمييز الكود

**مثال:**

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        warning: "hsl(var(--warning))",
        "warning-foreground": "hsl(var(--warning-foreground))",
      },
    },
  },
}
```

**الكود:**

````mdx
```js
module.exports = {
  theme: {
    extend: {
      colors: {
        warning: "hsl(var(--warning))",
        "warning-foreground": "hsl(var(--warning-foreground))",
      },
    },
  },
}
```
````

## تمييز الكود داخل السطر

**مثال:**

نتيجة `[1, 2, 3].join('-'){:js}` هي `'1-2-3'{:js}`.

**الكود:**

```mdx
The result of `[1, 2, 3].join('-'){:js}` is `'1-2-3'{:js}`.
```

## تمييز الكود داخل السطر مع فهم السياق

على سبيل المثال، إذا كان لديك مقطع الكود التالي:

```js
function getStringLength(str) {
  return str.length;
}
```

عندما نشير إلى `getStringLength{:.entity.name.function}` بوصفها دالة،
يمكننا تلوينها على هذا الأساس. وينطبق الأمر نفسه على `function{:.keyword}`، أو
`str{:.variable.parameter}` مقابل `str{:.variable.other.object}`، إلخ. يتيح ذلك
ربط الكود داخل السطر دلالياً بأقرب كتلة كود يشير إليها.

**الكود:**
````mdx
When we refer to `getStringLength{:.entity.name.function}` as a function,
we can color it as such. Same with `function{:.keyword}`, or
`str{:.variable.parameter}` vs. `str{:.variable.other.object}`, etc. This allows
semantically link inline code to the nearest block of code it refers to.
````

## تمييز مجموعات الكلمات حسب المعرّف

ضع معرّفاً بعد `#` بعد الكلمات.
يتيح لك ذلك تلوين الأحرف بشكل مختلف اعتماداً على المعرّف المحدد.

**مثال:**

```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```

**الكود:**
````md
```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```
````

لتغيير الأنماط أو إضافة أنماط جديدة:

```css title="apps/web/src/styles/mdx.css"
[data-chars-id='v'] {
  @apply !text-pink-300 bg-rose-800/50 font-bold;
}
```
