تُحقن المكونات تلقائياً في كل ملفات MDX، من دون الحاجة إلى import يدوي.
هذا يعني أنك تستطيع استخدام أي مكون متاح!

لمعرفة المزيد عن المكونات المتاحة، أو لإضافتها أو تغييرها، راجع ملف `src/components/docs/mdx.tsx`.

## Accordion

**مثال:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      سؤال تجريبي يوضح كيف يظهر عنوان عنصر Accordion داخل مستند MDX.
    </AccordionTrigger>

    <AccordionContent>
      هذا محتوى عربي تجريبي داخل عنصر Accordion. يمكنك استخدام هذه المساحة لشرح إجابة طويلة، أو تقسيم المعلومات إلى فقرات صغيرة، أو عرض تفاصيل إضافية لا تريد إظهارها مباشرة في الصفحة.

      يمكن أن يحتوي المحتوى على أكثر من فقرة، ويظل المكون مسؤولاً عن إظهارها وإخفائها بطريقة تفاعلية داخل الوثائق.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      سؤال آخر يوضح إمكانية وجود أكثر من عنصر داخل Accordion نفسه.
    </AccordionTrigger>

    <AccordionContent>
      هذا مثال قصير على محتوى العنصر الثاني.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## التنبيهات

**مثال:**

<Alert className="my-4">
  <AlertTitle>تنبيه افتراضي</AlertTitle>

  <AlertDescription>
    هذا تنبيه افتراضي.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>تنبيه تحذيري</AlertTitle>

  <AlertDescription>
    هذا تنبيه تحذيري.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>تنبيه خطأ</AlertTitle>

  <AlertDescription>
    هذا تنبيه خطأ.
  </AlertDescription>
</Alert>

**الكود:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**مثال:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">التبويب 1</TabsTrigger>
    <TabsTrigger value="tab-2">التبويب 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    محتوى التبويب 1
  </TabsContent>

  <TabsContent value="tab-2">
    محتوى التبويب 2
  </TabsContent>
</Tabs>

**الكود:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## الخطوات

**مثال:**

<Steps>
  <Step>الخطوة 1:</Step>

  ```bash
  npx something
  ```

  <Step>الخطوة 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**الكود:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
