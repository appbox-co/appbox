MDX ফাইলে code handle করতে `opendocs` [`rehype-pretty-code`](https://github.com/atomiks/rehype-pretty-code) library ব্যবহার করে।
এর ফলে code কীভাবে দেখানো হবে তা আপনি অনেকভাবে customize করতে পারেন!

আরও তথ্যের জন্য official documentation এবং example দেখুন!

## Context-aware inline code highlighting

উদাহরণস্বরূপ, আপনার কাছে নিচের code block থাকলে:

```js
function getStringLength(str) {
  return str.length;
}
```

যখন আমরা `getStringLength{:.entity.name.function}`-কে function হিসেবে refer করি,
তখন সেটিকে সেই অনুযায়ী color করা যায়। `function{:.keyword}`-এর ক্ষেত্রেও একই, অথবা
`str{:.variable.parameter}` বনাম `str{:.variable.other.object}` ইত্যাদি। এতে inline code-কে তার কাছের যে code block-কে refer করে তার সঙ্গে semantically link করা যায়।

**কোড:**
````mdx
When we refer to `getStringLength{:.entity.name.function}` as a function,
we can color it as such. Same with `function{:.keyword}`, or
`str{:.variable.parameter}` vs. `str{:.variable.other.object}`, etc. This allows
semantically link inline code to the nearest block of code it refers to.
````

## id দিয়ে word group highlighting

wordগুলোর পরে `#` দিয়ে একটি id দিন।
এতে দেওয়া id অনুযায়ী character আলাদা রঙে দেখানো যায়।

**উদাহরণ:**

```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```

**কোড:**
````md
```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```
````

style বদলাতে বা নতুন style যোগ করতে:

```css title="apps/web/src/styles/mdx.css"
[data-chars-id='v'] {
  @apply !text-pink-300 bg-rose-800/50 font-bold;
}
```
