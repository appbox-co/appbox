Component সব MDX ফাইলে automatically inject করা হয়, তাই manual import দরকার হয় না।
এর অর্থ, available যেকোনো component আপনি ব্যবহার করতে পারেন!

available component সম্পর্কে আরও জানতে, নতুন component যোগ করতে বা বিদ্যমান component পরিবর্তন করতে `src/components/docs/mdx.tsx` ফাইল দেখুন।

## Alerts

**উদাহরণ:**

<Alert className="my-4">
  <AlertTitle>ডিফল্ট alert</AlertTitle>

  <AlertDescription>
    এটি একটি ডিফল্ট alert।
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>সতর্কতা alert</AlertTitle>

  <AlertDescription>
    এটি একটি সতর্কতা alert।
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>ত্রুটি alert</AlertTitle>

  <AlertDescription>
    এটি একটি ত্রুটি alert।
  </AlertDescription>
</Alert>

**কোড:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**উদাহরণ:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">ট্যাব ১</TabsTrigger>
    <TabsTrigger value="tab-2">ট্যাব ২</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    ট্যাব ১-এর content
  </TabsContent>

  <TabsContent value="tab-2">
    ট্যাব ২-এর content
  </TabsContent>
</Tabs>

**কোড:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**উদাহরণ:**

<Steps>
  <Step>ধাপ ১:</Step>

  ```bash
  npx something
  ```

  <Step>ধাপ ২:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**কোড:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
