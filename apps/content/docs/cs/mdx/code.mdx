`opendocs` používá knihovnu [`rehype-pretty-code`](https://github.com/atomiks/rehype-pretty-code)
ke zpracování kódu v souborech MDX. Díky tomu můžete výrazně přizpůsobit způsob, jakým se kód zobrazuje!

Další informace najdete v oficiální dokumentaci a příkladech.

## Nadpisy

**Příklad:**

```js title="apps/web/src/index.tsx"
import { App } from './App';
```

**Kód:**

````mdx
```js title="apps/web/src/index.tsx"
import { App } from './App';
```
````

```js

    extend: {
      colors: {
        warning: "hsl(var(--warning))",
        "warning-foreground": "hsl(var(--warning-foreground))",
```

## Zvýraznění inline kódu

**Příklad:**

Výsledek `[1, 2, 3].join('-'){:js}` je `'1-2-3'{:js}`.

**Kód:**

```mdx
The result of `[1, 2, 3].join('-'){:js}` is `'1-2-3'{:js}`.
```

## Kontextové zvýraznění inline kódu

Například pokud máte následující blok kódu:

```js
function getStringLength(str) {
  return str.length;
}
```

Když o `getStringLength{:.entity.name.function}` mluvíme jako o funkci,
můžeme ji tak i barevně označit. Stejně tak `function{:.keyword}` nebo
`str{:.variable.parameter}` oproti `str{:.variable.other.object}` atd. To umožňuje
sémanticky propojit inline kód s nejbližším blokem kódu, na který odkazuje.

**Kód:**
````mdx
When we refer to `getStringLength{:.entity.name.function}` as a function,
we can color it as such. Same with `function{:.keyword}`, or
`str{:.variable.parameter}` vs. `str{:.variable.other.object}`, etc. This allows
semantically link inline code to the nearest block of code it refers to.
````

## Zvýraznění skupin slov podle id

Za slova vložte id za znak `#`.
To umožňuje obarvit znaky odlišně podle zadaného id.

**Příklad:**

```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```

**Kód:**
````md
```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```
````

Jak změnit nebo přidat nové styly:

```css title="apps/web/src/styles/mdx.css"
[data-chars-id='v'] {
  @apply !text-pink-300 bg-rose-800/50 font-bold;
}
```
