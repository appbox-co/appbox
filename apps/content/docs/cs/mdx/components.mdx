Komponenty se automaticky vkládají do všech souborů MDX bez nutnosti ručního importu.
To znamená, že můžete použít jakoukoli dostupnou komponentu.

Další informace o dostupných komponentách, jejich přidání nebo změně najdete v souboru `src/components/docs/mdx.tsx`

## Alerts

**Příklad:**

<Alert className="my-4">
  <AlertTitle>Výchozí upozornění</AlertTitle>

  <AlertDescription>
    Toto je výchozí upozornění.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Varovné upozornění</AlertTitle>

  <AlertDescription>
    Toto je varovné upozornění.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Chybové upozornění</AlertTitle>

  <AlertDescription>
    Toto je chybové upozornění.
  </AlertDescription>
</Alert>

**Kód:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Příklad:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Karta 1</TabsTrigger>
    <TabsTrigger value="tab-2">Karta 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Obsah karty 1
  </TabsContent>

  <TabsContent value="tab-2">
    Obsah karty 2
  </TabsContent>
</Tabs>

**Kód:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Příklad:**

<Steps>
  <Step>Krok 1:</Step>

  ```bash
  npx something
  ```

  <Step>Krok 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kód:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
