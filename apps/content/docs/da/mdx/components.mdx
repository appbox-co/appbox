Komponenter injiceres automatisk i alle MDX-filer uden behov for manuel import.
Det betyder, at du kan bruge enhver komponent, der er tilgængelig!

For at lære mere om, tilføje eller ændre de tilgængelige komponenter, se filen `src/components/docs/mdx.tsx`

## Alerts

**Eksempel:**

<Alert className="my-4">
  <AlertTitle>Standardadvarsel</AlertTitle>

  <AlertDescription>
    Dette er en standardadvarsel.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Advarsel</AlertTitle>

  <AlertDescription>
    Dette er en advarsel.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Fejladvarsel</AlertTitle>

  <AlertDescription>
    Dette er en fejladvarsel.
  </AlertDescription>
</Alert>

**Kode:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Eksempel:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Fane 1</TabsTrigger>
    <TabsTrigger value="tab-2">Fane 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Indhold for fane 1
  </TabsContent>

  <TabsContent value="tab-2">
    Indhold for fane 2
  </TabsContent>
</Tabs>

**Kode:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Eksempel:**

<Steps>
  <Step>Trin 1:</Step>

  ```bash
  npx something
  ```

  <Step>Trin 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kode:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
