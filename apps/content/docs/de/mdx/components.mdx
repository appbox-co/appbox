Komponenten werden automatisch, ohne die Notwendigkeit eines manuellen Imports, in alle MDX-Dateien eingefügt.
Das bedeutet, ihr könnt jede verfügbare Komponente verwenden!

Um mehr über die verfügbaren Komponenten zu erfahren, sie hinzuzufügen oder zu ändern, siehe die Datei `src/components/docs/mdx.tsx`

## Alerts

**Beispiel:**

<Alert className="my-4">
  <AlertTitle>Standard Alert</AlertTitle>

  <AlertDescription>
    Dies ist ein Standard Alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warnung Alert</AlertTitle>

  <AlertDescription>
    Dies ist ein Warnungs Alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Fehler Alert</AlertTitle>

  <AlertDescription>
    Dies ist ein Fehler Alert.
  </AlertDescription>
</Alert>

    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>

  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 Inhalt
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 Inhalt

**Beispiel:**

  <Step>Schritt 1:</Step>

  ```bash
  npx something
  ```

  <Step>Schritt 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
