Τα components εισάγονται αυτόματα σε όλα τα αρχεία MDX, χωρίς να απαιτείται χειροκίνητο import.
Αυτό σημαίνει ότι μπορείτε να χρησιμοποιήσετε οποιοδήποτε διαθέσιμο component!

Για να μάθετε περισσότερα, να προσθέσετε ή να αλλάξετε τα διαθέσιμα components, δείτε το αρχείο `src/components/docs/mdx.tsx`

## Accordion

**Παράδειγμα:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Αυτό είναι ένα παράδειγμα ερώτησης για το πρώτο στοιχείο του accordion.
    </AccordionTrigger>

    <AccordionContent>
      Αυτό είναι ενδεικτικό περιεχόμενο για το πρώτο στοιχείο του accordion. Μπορεί να περιλαμβάνει μεγαλύτερες παραγράφους, επεξηγήσεις και λεπτομέρειες που ο χρήστης βλέπει αφού ανοίξει το στοιχείο. Χρησιμοποιήστε το για να παρουσιάσετε πληροφορίες με συμπτυσσόμενο τρόπο.

      Μπορείτε επίσης να προσθέσετε δεύτερη παράγραφο με πρόσθετες λεπτομέρειες, παραδείγματα ή σημειώσεις που σχετίζονται με την ίδια ερώτηση.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Αυτό είναι ένα δεύτερο παράδειγμα ερώτησης μέσα στο accordion.
    </AccordionTrigger>

    <AccordionContent>
      Αυτό είναι σύντομο ενδεικτικό περιεχόμενο για το δεύτερο στοιχείο.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Alerts

**Παράδειγμα:**

<Alert className="my-4">
  <AlertTitle>Προεπιλεγμένο alert</AlertTitle>

  <AlertDescription>
    Αυτό είναι ένα προεπιλεγμένο alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Προειδοποιητικό alert</AlertTitle>

  <AlertDescription>
    Αυτό είναι ένα προειδοποιητικό alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alert σφάλματος</AlertTitle>

  <AlertDescription>
    Αυτό είναι ένα alert σφάλματος.
  </AlertDescription>
</Alert>

**Κώδικας:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Παράδειγμα:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Καρτέλα 1</TabsTrigger>
    <TabsTrigger value="tab-2">Καρτέλα 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Περιεχόμενο καρτέλας 1
  </TabsContent>

  <TabsContent value="tab-2">
    Περιεχόμενο καρτέλας 2
  </TabsContent>
</Tabs>

**Κώδικας:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Παράδειγμα:**

<Steps>
  <Step>Βήμα 1:</Step>

  ```bash
  npx something
  ```

  <Step>Βήμα 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

  ```bash

  ```

  ```bash

  ```
