Komponentit injektoidaan automaattisesti kaikkiin MDX-tiedostoihin ilman manuaalista importia.
Tämä tarkoittaa, että voit käyttää mitä tahansa saatavilla olevaa komponenttia!

Jos haluat oppia lisää saatavilla olevista komponenteista, lisätä niitä tai muuttaa niitä, katso tiedosto `src/components/docs/mdx.tsx`

## Hälytykset

**Esimerkki:**

<Alert className="my-4">
  <AlertTitle>Oletushälytys</AlertTitle>

  <AlertDescription>
    Tämä on oletushälytys.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Varoitushälytys</AlertTitle>

  <AlertDescription>
    Tämä on varoitushälytys.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Virhehälytys</AlertTitle>

  <AlertDescription>
    Tämä on virhehälytys.
  </AlertDescription>
</Alert>

**Koodi:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Välilehdet

**Esimerkki:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Välilehti 1</TabsTrigger>
    <TabsTrigger value="tab-2">Välilehti 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Välilehden 1 sisältö
  </TabsContent>

  <TabsContent value="tab-2">
    Välilehden 2 sisältö
  </TabsContent>
</Tabs>

**Koodi:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Vaiheet

**Esimerkki:**

<Steps>
  <Step>Vaihe 1:</Step>

  ```bash
  npx something
  ```

  <Step>Vaihe 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Koodi:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
