Les composants sont automatiquement injectés dans tous les fichiers MDX, sans import manuel.
Cela signifie que vous pouvez utiliser n'importe quel composant disponible !

Pour en savoir plus, ajouter ou modifier les composants disponibles, consultez le fichier `src/components/docs/mdx.tsx`

## Alertes

**Exemple :**

<Alert className="my-4">
  <AlertTitle>Alerte par défaut</AlertTitle>

  <AlertDescription>
    Ceci est une alerte par défaut.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alerte d'avertissement</AlertTitle>

  <AlertDescription>
    Ceci est une alerte d'avertissement.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alerte d'erreur</AlertTitle>

  <AlertDescription>
    Ceci est une alerte d'erreur.
  </AlertDescription>
</Alert>

**Code :**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Onglets

**Exemple :**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Onglet 1</TabsTrigger>
    <TabsTrigger value="tab-2">Onglet 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Contenu de l'onglet 1
  </TabsContent>

  <TabsContent value="tab-2">
    Contenu de l'onglet 2
  </TabsContent>
</Tabs>

**Code :**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Étapes

**Exemple :**

<Steps>
  <Step>Étape 1 :</Step>

  ```bash
  npx something
  ```

  <Step>Étape 2 :</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Code :**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
