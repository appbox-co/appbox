## Isticanje inline koda

**Primjer:**

Rezultat izraza `[1, 2, 3].join('-'){:js}` je `'1-2-3'{:js}`.

**Kod:**

```mdx

```

## Kontekstualno isticanje inline koda

Na primjer, ako imate sljedeći blok koda:

```js
function getStringLength(str) {
  return str.length;
}
```

Kada `getStringLength{:.entity.name.function}` spominjemo kao funkciju,
možemo ga tako i obojiti. Isto vrijedi za `function{:.keyword}` ili
`str{:.variable.parameter}` nasuprot `str{:.variable.other.object}` itd. To omogućuje
semantičko povezivanje inline koda s najbližim blokom koda na koji se odnosi.

**Kod:**
````mdx
When we refer to `getStringLength{:.entity.name.function}` as a function,
we can color it as such. Same with `function{:.keyword}`, or
`str{:.variable.parameter}` vs. `str{:.variable.other.object}`, etc. This allows
semantically link inline code to the nearest block of code it refers to.
````

## Isticanje grupa riječi prema id-u

Stavite id nakon `#` iza riječi.
To vam omogućuje da znakove obojite drukčije na temelju zadanog id-a.

**Primjer:**

```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```

**Kod:**
````md
```js /age/#v /name/#v /setAge/#s /setName/#s /50/#i /"Taylor"/#i
const [age, setAge] = useState(50);
const [name, setName] = useState("Taylor");
```
````

Za promjenu ili dodavanje novih stilova:

```css title="apps/web/src/styles/mdx.css"
[data-chars-id='v'] {
  @apply !text-pink-300 bg-rose-800/50 font-bold;
}
```
