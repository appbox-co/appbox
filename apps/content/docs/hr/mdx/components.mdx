Komponente se automatski ubacuju u sve MDX datoteke, bez potrebe za ručnim importom.
To znači da možete koristiti bilo koju dostupnu komponentu!

Da biste saznali više, dodali ili promijenili dostupne komponente, pogledajte datoteku `src/components/docs/mdx.tsx`

## Accordion

**Primjer:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Ovo je prvi primjer pitanja za Accordion komponentu.
    </AccordionTrigger>

    <AccordionContent>
      Ovo je primjer sadržaja unutar Accordion stavke. Možete ga koristiti za odgovore na česta pitanja, dodatna objašnjenja ili bilo koji sadržaj koji želite prikazati tek nakon otvaranja stavke.

      Drugi odlomak pokazuje kako komponenta može sadržavati više odlomaka i dulji tekst, uz zadržavanje uredne strukture dokumenta.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Ovo je drugi primjer pitanja za Accordion komponentu.
    </AccordionTrigger>

    <AccordionContent>
      Ovo je kraći primjer odgovora za drugu stavku.
    </AccordionContent>
  </AccordionItem>
</Accordion>

      Velit dolore quis veniam ex aliqua. Et velit occaecat ea aliquip id quis voluptate ipsum culpa eu enim quis amet tempor. Minim cillum deserunt quis consequat esse eu irure sunt dolore nisi dolor sit adipisicing. Dolor aliqua nulla sit culpa incididunt magna commodo voluptate qui labore cillum consequat consequat.

## Alerts

**Primjer:**

<Alert className="my-4">
  <AlertTitle>Zadani alert</AlertTitle>

  <AlertDescription>
    Ovo je zadani alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alert upozorenja</AlertTitle>

  <AlertDescription>
    Ovo je alert upozorenja.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alert pogreške</AlertTitle>

  <AlertDescription>
    Ovo je alert pogreške.
  </AlertDescription>
</Alert>

**Kod:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Primjer:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Kartica 1</TabsTrigger>
    <TabsTrigger value="tab-2">Kartica 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Sadržaj kartice 1
  </TabsContent>

  <TabsContent value="tab-2">
    Sadržaj kartice 2
  </TabsContent>
</Tabs>

**Kod:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Primjer:**

<Steps>
  <Step>Korak 1:</Step>

  ```bash
  npx something
  ```

  <Step>Korak 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kod:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
