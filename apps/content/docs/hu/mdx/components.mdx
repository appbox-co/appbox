A komponensek automatikusan bekerülnek minden MDX fájlba, kézi importálás nélkül.
Ez azt jelenti, hogy bármely elérhető komponenst használhatod!

Az elérhető komponensekről, illetve azok hozzáadásáról vagy módosításáról a `src/components/docs/mdx.tsx` fájlban tudhatsz meg többet.

## Harmonika

**Példa:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Ez az első gyakran ismételt kérdés címsora.
    </AccordionTrigger>

    <AccordionContent>
      Ez az első válasz tartalma. Használhatod hosszabb magyarázatokhoz, részletes leírásokhoz vagy olyan információkhoz, amelyeket csak akkor szeretnél megjeleníteni, amikor a felhasználó megnyitja az elemet.

      Több bekezdést is tartalmazhat, így jól használható összetettebb válaszokhoz és részletesebb útmutatókhoz.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Ez a második gyakran ismételt kérdés címsora.
    </AccordionTrigger>

    <AccordionContent>
      Ez a második válasz rövidebb tartalma.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Figyelmeztetések

**Példa:**

<Alert className="my-4">
  <AlertTitle>Alapértelmezett figyelmeztetés</AlertTitle>

  <AlertDescription>
    Ez egy alapértelmezett figyelmeztetés.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Figyelmeztető üzenet</AlertTitle>

  <AlertDescription>
    Ez egy figyelmeztető üzenet.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Hibaüzenet</AlertTitle>

  <AlertDescription>
    Ez egy hibaüzenet.
  </AlertDescription>
</Alert>

**Kód:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Fülek

**Példa:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">1. fül</TabsTrigger>
    <TabsTrigger value="tab-2">2. fül</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Az 1. fül tartalma
  </TabsContent>

  <TabsContent value="tab-2">
    A 2. fül tartalma
  </TabsContent>
</Tabs>

**Kód:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```
