Komponen otomatis diinjeksi ke semua file MDX tanpa perlu import manual.
Artinya Anda dapat menggunakan komponen apa pun yang tersedia!

Untuk mempelajari lebih lanjut, menambahkan, atau mengubah komponen yang tersedia, lihat file `src/components/docs/mdx.tsx`

## Alerts

**Contoh:**

<Alert className="my-4">
  <AlertTitle>Alert default</AlertTitle>

  <AlertDescription>
    Ini adalah alert default.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alert peringatan</AlertTitle>

  <AlertDescription>
    Ini adalah alert peringatan.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alert error</AlertTitle>

  <AlertDescription>
    Ini adalah alert error.
  </AlertDescription>
</Alert>

**Kode:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Contoh:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Konten Tab 1
  </TabsContent>

  <TabsContent value="tab-2">
    Konten Tab 2
  </TabsContent>
</Tabs>

**Kode:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Contoh:**

<Steps>
  <Step>Langkah 1:</Step>

  ```bash
  npx something
  ```

  <Step>Langkah 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kode:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
