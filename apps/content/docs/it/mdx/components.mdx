I componenti vengono iniettati automaticamente in tutti i file MDX, senza bisogno di importarli manualmente.
Questo significa che puoi usare qualsiasi componente disponibile.

Per saperne di più, aggiungere o modificare i componenti disponibili, consulta il file `src/components/docs/mdx.tsx`

## Avvisi

**Esempio:**

<Alert className="my-4">
  <AlertTitle>Avviso predefinito</AlertTitle>

  <AlertDescription>
    Questo è un avviso predefinito.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Avviso di avvertimento</AlertTitle>

  <AlertDescription>
    Questo è un avviso di avvertimento.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Avviso di errore</AlertTitle>

  <AlertDescription>
    Questo è un avviso di errore.
  </AlertDescription>
</Alert>

**Codice:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Schede

**Esempio:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Scheda 1</TabsTrigger>
    <TabsTrigger value="tab-2">Scheda 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Contenuto scheda 1
  </TabsContent>

  <TabsContent value="tab-2">
    Contenuto scheda 2
  </TabsContent>
</Tabs>

**Codice:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Passaggi

**Esempio:**

<Steps>
  <Step>Passaggio 1:</Step>

  ```bash
  npx something
  ```

  <Step>Passaggio 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Codice:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
