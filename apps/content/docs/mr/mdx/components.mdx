Components सर्व MDX files मध्ये manual import शिवाय automatically injected केले जातात.
याचा अर्थ उपलब्ध असलेला कोणताही component तुम्ही वापरू शकता.

उपलब्ध components बद्दल अधिक जाणून घेण्यासाठी, त्यात भर घालण्यासाठी किंवा बदल करण्यासाठी `src/components/docs/mdx.tsx` फाइल पहा.

## Accordion

**उदाहरण:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      हे पहिले accordion item उघडण्यासाठीचे sample प्रश्न किंवा शीर्षक आहे.
    </AccordionTrigger>

    <AccordionContent>
      हा accordion content चा नमुना आहे. येथे तुम्ही वापरकर्त्याला दिसणारी अतिरिक्त माहिती, स्पष्टीकरण, किंवा सविस्तर उत्तर लिहू शकता. मजकूर मोठा असू शकतो आणि component त्याला व्यवस्थित दाखवतो.

      दुसरा परिच्छेदही ठेवता येतो, ज्यामुळे content spacing आणि formatting तपासता येते.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      हे दुसरे accordion item आहे.
    </AccordionTrigger>

    <AccordionContent>
      दुसऱ्या item साठीचा छोटा content नमुना.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Alerts

**उदाहरण:**

<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    हा default alert आहे.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    हा warning alert आहे.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    हा error alert आहे.
  </AlertDescription>
</Alert>

**Code:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**उदाहरण:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>

**Code:**

```jsx
<Tabs defaultValue="tab-1">

    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>

  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>

```
