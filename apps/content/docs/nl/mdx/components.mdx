Componenten worden automatisch in alle MDX-bestanden geinjecteerd, zonder dat je ze handmatig hoeft te importeren.
Dit betekent dat je elk beschikbaar component kunt gebruiken!

Zie het bestand `src/components/docs/mdx.tsx` om meer te leren over beschikbare componenten, of om ze toe te voegen of te wijzigen.

## Alerts

**Voorbeeld:**

<Alert className="my-4">
  <AlertTitle>Standaardalert</AlertTitle>

  <AlertDescription>
    Dit is een standaardalert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Waarschuwingsalert</AlertTitle>

  <AlertDescription>
    Dit is een waarschuwingsalert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Foutalert</AlertTitle>

  <AlertDescription>
    Dit is een foutalert.
  </AlertDescription>
</Alert>

**Code:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Voorbeeld:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Content van tab 1
  </TabsContent>

  <TabsContent value="tab-2">
    Content van tab 2
  </TabsContent>
</Tabs>

**Code:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Voorbeeld:**

<Steps>
  <Step>Stap 1:</Step>

  ```bash
  npx something
  ```

  <Step>Stap 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Code:**

````jsx
<Steps>

  ```bash

  npx something-else
  ```
````
