Komponenter injiseres automatisk i alle MDX-filer, uten behov for manuell import.
Dette betyr at du kan bruke alle komponenter som er tilgjengelige!

For å lære mer om, legge til eller endre de tilgjengelige komponentene, se filen `src/components/docs/mdx.tsx`

## Varsler

**Eksempel:**

<Alert className="my-4">
  <AlertTitle>Standardvarsel</AlertTitle>

  <AlertDescription>
    Dette er et standardvarsel.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Advarselsvarsel</AlertTitle>

  <AlertDescription>
    Dette er et advarselsvarsel.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Feilvarsel</AlertTitle>

  <AlertDescription>
    Dette er et feilvarsel.
  </AlertDescription>
</Alert>

**Kode:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Eksempel:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Fane 1</TabsTrigger>
    <TabsTrigger value="tab-2">Fane 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Innhold i fane 1
  </TabsContent>

  <TabsContent value="tab-2">
    Innhold i fane 2
  </TabsContent>
</Tabs>

**Kode:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Trinn

**Eksempel:**

<Steps>
  <Step>Trinn 1:</Step>

  ```bash
  npx something
  ```

  <Step>Trinn 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kode:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
