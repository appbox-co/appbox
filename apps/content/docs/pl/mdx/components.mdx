Komponenty są automatycznie wstrzykiwane do wszystkich plików MDX, bez potrzeby ręcznego importowania.
Oznacza to, że możesz używać dowolnego dostępnego komponentu!

Aby dowiedzieć się więcej, dodać lub zmienić dostępne komponenty, zobacz plik `src/components/docs/mdx.tsx`

## Accordion

**Przykład:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Przykładowe pytanie dotyczące działania elementu Accordion.
    </AccordionTrigger>

    <AccordionContent>
      To jest przykładowa treść pierwszego elementu Accordion. Możesz umieścić tutaj dłuższy opis, instrukcję albo odpowiedź, która będzie rozwijana po kliknięciu nagłówka.

      Drugi akapit pokazuje, że zawartość może obejmować kilka fragmentów tekstu i nadal pozostaje częścią tego samego panelu.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Kolejne przykładowe pytanie w komponencie Accordion.
    </AccordionTrigger>

    <AccordionContent>
      To jest krótka przykładowa odpowiedź dla drugiego elementu.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Alerty

**Przykład:**

<Alert className="my-4">
  <AlertTitle>Alert domyślny</AlertTitle>

  <AlertDescription>
    To jest alert domyślny.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alert ostrzegawczy</AlertTitle>

  <AlertDescription>
    To jest alert ostrzegawczy.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alert błędu</AlertTitle>

  <AlertDescription>
    To jest alert błędu.
  </AlertDescription>
</Alert>

**Kod:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Karty

**Przykład:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Karta 1</TabsTrigger>
    <TabsTrigger value="tab-2">Karta 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Treść karty 1
  </TabsContent>

  <TabsContent value="tab-2">
    Treść karty 2
  </TabsContent>
</Tabs>

**Kod:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Kroki

**Przykład:**

<Steps>
  <Step>Krok 1:</Step>

  ```bash
  npx something
  ```

  <Step>Krok 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kod:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
