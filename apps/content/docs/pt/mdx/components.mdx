Os componentes são injetados automaticamente em todos os arquivos MDX, sem necessidade de importação manual.
Isso significa que você pode usar qualquer componente disponível.

Para saber mais, adicionar ou alterar os componentes disponíveis, consulte o arquivo `src/components/docs/mdx.tsx`.

## Alertas

**Exemplo:**

<Alert className="my-4">
  <AlertTitle>Alerta padrão</AlertTitle>

  <AlertDescription>
    Este é um alerta padrão.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alerta de aviso</AlertTitle>

  <AlertDescription>
    Este é um alerta de aviso.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alerta de erro</AlertTitle>

  <AlertDescription>
    Este é um alerta de erro.
  </AlertDescription>
</Alert>

**Código:**

```jsx
<Alert className="my-4">
  <AlertTitle>Alerta padrão</AlertTitle>

  <AlertDescription>
    Este é um alerta padrão.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alerta de aviso</AlertTitle>

  <AlertDescription>
    Este é um alerta de aviso.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alerta de erro</AlertTitle>

  <AlertDescription>
    Este é um alerta de erro.
  </AlertDescription>
</Alert>
```

## Abas

**Exemplo:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Aba 1</TabsTrigger>
    <TabsTrigger value="tab-2">Aba 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Conteúdo da Aba 1
  </TabsContent>

  <TabsContent value="tab-2">
    Conteúdo da Aba 2
  </TabsContent>
</Tabs>

**Código:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Aba 1</TabsTrigger>
    <TabsTrigger value="tab-2">Aba 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Conteúdo da Aba 1
  </TabsContent>

  <TabsContent value="tab-2">
    Conteúdo da Aba 2
  </TabsContent>
</Tabs>
```

## Etapas

**Exemplo:**

<Steps>
  <Step>Etapa 1:</Step>

  ```bash
  npx something
  ```

  <Step>Etapa 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Código:**

````jsx
<Steps>
  <Step>Etapa 1:</Step>

  ```bash
  npx something
  ```

  <Step>Etapa 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
