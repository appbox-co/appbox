Componentele sunt injectate automat, fără a fi nevoie de import manual, în toate fișierele MDX.
Aceasta înseamnă că poți folosi orice componentă disponibilă!

Pentru a afla mai multe despre componentele disponibile, pentru a le adăuga sau modifica, consultă fișierul `src/components/docs/mdx.tsx`

## Alerte

**Exemplu:**

<Alert className="my-4">
  <AlertTitle>Alertă implicită</AlertTitle>

  <AlertDescription>
    Aceasta este o alertă implicită.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alertă de avertizare</AlertTitle>

  <AlertDescription>
    Aceasta este o alertă de avertizare.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alertă de eroare</AlertTitle>

  <AlertDescription>
    Aceasta este o alertă de eroare.
  </AlertDescription>
</Alert>

**Cod:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## File

**Exemplu:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Fila 1</TabsTrigger>
    <TabsTrigger value="tab-2">Fila 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Conținut fila 1
  </TabsContent>

  <TabsContent value="tab-2">
    Conținut fila 2
  </TabsContent>
</Tabs>

**Cod:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Pași

**Exemplu:**

<Steps>
  <Step>Pasul 1:</Step>

  ```bash
  npx something
  ```

  <Step>Pasul 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Cod:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
