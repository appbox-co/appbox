Компоненте се аутоматски убацују у све MDX датотеке, без потребе за ручним import-ом.
То значи да можете користити било коју компоненту која је доступна!

Да бисте сазнали више, додали или променили доступне компоненте, погледајте датотеку `src/components/docs/mdx.tsx`

## Alerts

**Пример:**

<Alert className="my-4">
  <AlertTitle>Подразумевано упозорење</AlertTitle>

  <AlertDescription>
    Ово је подразумевано упозорење.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Упозорење</AlertTitle>

  <AlertDescription>
    Ово је упозорење.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Упозорење о грешци</AlertTitle>

  <AlertDescription>
    Ово је упозорење о грешци.
  </AlertDescription>
</Alert>

**Код:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Пример:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Садржај Tab 1
  </TabsContent>

  <TabsContent value="tab-2">
    Садржај Tab 2
  </TabsContent>
</Tabs>

**Код:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Пример:**

<Steps>
  <Step>Корак 1:</Step>

  ```bash
  npx something
  ```

  <Step>Корак 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Код:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
