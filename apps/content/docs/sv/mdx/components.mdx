Komponenter injiceras automatiskt i alla MDX-filer, utan att de behöver importeras manuellt.
Det betyder att du kan använda alla komponenter som finns tillgängliga!

Om du vill lära dig mer om, lägga till eller ändra de tillgängliga komponenterna, se filen `src/components/docs/mdx.tsx`

## Aviseringar

**Exempel:**

<Alert className="my-4">
  <AlertTitle>Standardavisering</AlertTitle>

  <AlertDescription>
    Det här är en standardavisering.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Varningsavisering</AlertTitle>

  <AlertDescription>
    Det här är en varningsavisering.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Felavisering</AlertTitle>

  <AlertDescription>
    Det här är en felavisering.
  </AlertDescription>
</Alert>

**Kod:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Flikar

**Exempel:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Flik 1</TabsTrigger>
    <TabsTrigger value="tab-2">Flik 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Innehåll för flik 1
  </TabsContent>

  <TabsContent value="tab-2">
    Innehåll för flik 2
  </TabsContent>
</Tabs>

**Kod:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steg

**Exempel:**

<Steps>
  <Step>Steg 1:</Step>

  ```bash
  npx something
  ```

  <Step>Steg 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kod:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
