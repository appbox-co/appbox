Components అన్ని MDX ఫైల్‌లలో manual import అవసరం లేకుండా స్వయంచాలకంగా inject చేయబడతాయి.
దీని అర్థం అందుబాటులో ఉన్న ఏ component అయినా ఉపయోగించవచ్చు!

అందుబాటులో ఉన్న components గురించి మరింత తెలుసుకోవడానికి, వాటిని జోడించడానికి లేదా మార్చడానికి `src/components/docs/mdx.tsx` ఫైల్ చూడండి.

## Accordion component

**ఉదాహరణ:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Appbox లో files, apps, settings ఎలా కలిసి పనిచేస్తాయో తెలుసుకోవడానికి ఈ విభాగాన్ని తెరవండి.
    </AccordionTrigger>

    <AccordionContent>
      ఇక్కడ మీరు యూజర్‌కు చూపించే సహాయక వివరణను జోడించవచ్చు. ఇది పొడవైన content ను collapse చేయడానికి ఉపయోగపడుతుంది, తద్వారా పేజీ చదవడానికి సులభంగా ఉంటుంది.

      అవసరమైతే రెండో paragraph కూడా చేర్చవచ్చు. Accordion content MDX లో సాధారణ prose, inline code, మరియు ఇతర allowed components తో పనిచేస్తుంది.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      మరో తరచుగా అడిగే ప్రశ్నను ఇక్కడ ఉంచవచ్చు.
    </AccordionTrigger>

    <AccordionContent>
      రెండో సమాధానం ఇక్కడ కనిపిస్తుంది.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Alerts component

**ఉదాహరణ:**

<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    ఇది default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    ఇది warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    ఇది error alert.
  </AlertDescription>
</Alert>

## Tabs component

**ఉదాహరణ:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 కంటెంట్
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 కంటెంట్
  </TabsContent>
</Tabs>

**Code:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
```
