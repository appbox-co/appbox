Bileşenler, elle import gerektirmeden tüm MDX dosyalarına otomatik olarak enjekte edilir.
Bu, kullanılabilir olan herhangi bir bileşeni kullanabileceğiniz anlamına gelir!

Kullanılabilir bileşenler hakkında daha fazla bilgi edinmek, bileşen eklemek veya değiştirmek için `src/components/docs/mdx.tsx` dosyasına bakın.

## Accordion

**Örnek:**

<Accordion type="multiple">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      Bu, ilk sık sorulan soru için örnek bir accordion başlığıdır.
    </AccordionTrigger>

    <AccordionContent>
      Bu alanda accordion öğesinin ayrıntılı içeriği yer alır. MDX içinde bileşenleri kullanarak kullanıcıya ek açıklamalar, uzun yanıtlar veya gizlenebilir bilgi blokları gösterebilirsiniz. İçerik gerektiğinde birden fazla paragraf içerebilir.

      İkinci paragraf, accordion içeriğinin nasıl akacağını göstermek için eklenmiştir. Bu örnek, gerçek dokümantasyon metnini temsil eden yer tutucu bir açıklamadır.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Bu, ikinci örnek accordion başlığıdır.
    </AccordionTrigger>

    <AccordionContent>
      Bu ikinci accordion öğesinin kısa örnek içeriğidir.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Alerts

**Örnek:**

<Alert className="my-4">
  <AlertTitle>Varsayılan uyarı</AlertTitle>

  <AlertDescription>
    Bu varsayılan bir uyarıdır.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Uyarı bildirimi</AlertTitle>

  <AlertDescription>
    Bu bir uyarı bildirimidir.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Hata uyarısı</AlertTitle>

  <AlertDescription>
    Bu bir hata uyarısıdır.
  </AlertDescription>
</Alert>

**Kod:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**Örnek:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Sekme 1</TabsTrigger>
    <TabsTrigger value="tab-2">Sekme 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Sekme 1 içeriği
  </TabsContent>

  <TabsContent value="tab-2">
    Sekme 2 içeriği
  </TabsContent>
</Tabs>

**Kod:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>
```

## Steps

**Örnek:**

<Steps>
  <Step>Adım 1:</Step>

  ```bash
  npx something
  ```

  <Step>Adım 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Kod:**

````jsx
<Steps>
  <Step>Step 1:</Step>

  ```bash
  npx something
  ```

  <Step>Step 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
