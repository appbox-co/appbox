تمام MDX files میں components manual import کے بغیر automatically inject ہوتے ہیں۔
اس کا مطلب ہے کہ آپ کوئی بھی available component استعمال کر سکتے ہیں!

Available components کے بارے میں مزید جاننے، add کرنے یا change کرنے کے لیے `src/components/docs/mdx.tsx` file دیکھیں۔

## Alerts

**مثال:**

<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    یہ default alert ہے۔
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    یہ warning alert ہے۔
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    یہ error alert ہے۔
  </AlertDescription>
</Alert>

**Code:**

```jsx
<Alert className="my-4">
  <AlertTitle>Default alert</AlertTitle>

  <AlertDescription>
    This is a default alert.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Warning alert</AlertTitle>

  <AlertDescription>
    This is a warning alert.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Error alert</AlertTitle>

  <AlertDescription>
    This is an error alert.
  </AlertDescription>
</Alert>
```

## Tabs

**مثال:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab-2">Tab 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Tab 1 content
  </TabsContent>

  <TabsContent value="tab-2">
    Tab 2 content
  </TabsContent>
</Tabs>

**Code:**
