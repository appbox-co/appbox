আমাদের best efforts এবং rigorous audits সত্ত্বেও, security incidents যেকোনো online business-এর inevitable অংশ। আপনার security আমাদের কাছে গুরুত্বপূর্ণ, এবং আমরা আপনার security ও আমাদের platform-এর security উভয়ের প্রতি fully committed। আমরা transparency এবং honesty-র culture বজায় রাখি, যা security incidents report করার আমাদের approach-এও প্রযোজ্য। Security incident ঘটলে, Appbox নিম্নলিখিত actions নেবে:

1. প্রয়োজন হলে, incident দ্রুত resolve করতে affected Appbox servers-এ connections restrict করব
2. কোনো incidents reported বা discovered হওয়ার সঙ্গে সঙ্গে, আমরা আমাদের communication channels-এ details publish করব, যার মধ্যে থাকবে:
   - আমাদের website
   - আমাদের [status page](https://status.appbox.co)
   - Affected clients-কে email notifications

আমরা affected clients-কে সরাসরি email করাও শুরু করব, যদিও সব communications dispatch হতে কিছু সময় লাগতে পারে।

আপনি যদি আপনার services নিয়ে issues experience করেন, অনুগ্রহ করে আমাদের [status page](https://status.appbox.co) check করুন। সম্ভবত আপনি আপনার service সম্পর্কিত maintenance work দেখতে পাবেন। Ongoing maintenance না থাকলে, অনুগ্রহ করে [support ticket খুলুন](https://billing.appbox.co/submitticket.php?step=2&deptid=3), যাতে আমরা আপনার issues investigate করতে এবং accordingly guide করতে পারি।

## Security-First Infrastructure

* আমরা third-party partners এবং vendors-কে suitability এবং security risk উভয় দিক থেকে carefully assess করি। সব partnerships-এর জন্য সঠিক technical এবং contractual commitments আছে তা নিশ্চিত করি।
* আপনার availability এবং security মাথায় রেখে আমরা enterprise-grade, high-availability data centers ব্যবহার করি - সবই rigorous physical security measures সহ।
* Customer data isolate করতে, security enhance করতে, এবং potential network-level intrusions prevent করতে আমরা আমাদের network infrastructure-এর মধ্যে private VLANs implement করি। Optimal security নিশ্চিত করতে আমাদের in-house networking team এই VLANs manage করে।

## আপনি কীভাবে সাহায্য করতে পারেন

আমাদের [Terms of Service](/policies/terms-of-service) এবং security documentation জুড়ে বর্ণিত হিসাবে, Appbox secure রাখতে আমরা extensive measures নিই। Platform আরও protect করতে আপনি সাহায্য করতে পারেন:

* আমাদের [Password Policy](/policies/password-policy) follow করে
* Account sharing সম্পর্কে careful থেকে - যাদের আপনি চেনেন না বা trust করেন না তাদের সাথে কখনও account share করবেন না
* আপনার account-এ unauthorized access, suspicious password use সহ, notice করলে [Appbox-কে immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) করে
* আপনার account-এ two-factor authentication (2FA) enable করে

## Reporting Security Concerns

* **Security Events**: Unauthorized account usage বা suspected data breach-এর মতো security event report করতে অনুগ্রহ করে [Appbox-কে immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) করুন
* **Vulnerabilities**: Vulnerabilities report করা এবং potential rewards সম্পর্কে detailed information-এর জন্য আমাদের [Bug Bounty Program](/policies/bug-bounty-program) দেখুন
* **Sensitive Data**: Appbox থেকে sensitive data remove করতে হলে, অনুগ্রহ করে আমাদের [Sensitive Data Removal Policy](/policies/sensitive-data-removal-policy) দেখুন
* **Content Abuse**: Copyright বা content abuse issues কীভাবে report করবেন তা বুঝতে আমাদের [Content Abuse Policy](/policies/content-abuse-policy) পড়ুন
