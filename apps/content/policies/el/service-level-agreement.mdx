Το παρόν Service Level Agreement ("SLA") είναι έγγραφο που διέπει τους Όρους και τη χρήση των Υπηρεσιών που παρέχονται από το Appbox. Χρησιμοποιώντας τις Υπηρεσίες μας, αναγνωρίζετε και συναινείτε στους όρους που ορίζονται παρακάτω.

Επειδή λειτουργούμε σε shared environment και παρέχουμε υπηρεσίες συγκεκριμένης φύσης, κάποιο επίπεδο downtime μπορεί μερικές φορές να είναι αναπόφευκτο. Η παρούσα συμφωνία διευκρινίζει πώς διαχειριζόμαστε περιστατικά downtime κατά τη διάρκεια της περιόδου συνδρομής σας και περιγράφει τις δεσμεύσεις μας για επίλυση διαφορών και προσφορά αποζημίωσης.

## Server downtimes, server maintenance και compensation

Το Appbox καταβάλλει κάθε προσπάθεια ώστε οι servers μας να παραμένουν λειτουργικοί και προσβάσιμοι για όλους τους χρήστες καθ' όλη τη διάρκεια του billing term τους.

Όταν προγραμματίζεται server maintenance εκ των προτέρων, οι χρήστες θα ειδοποιούνται μέσω πολλαπλών καναλιών - όπως email updates, Appbox Status Page και Client Area Announcements. Οποιοσδήποτε χρήστης επηρεάζεται από planned maintenance δεν είναι επιλέξιμος για compensation.

Οι μη προγραμματισμένες ή emergency maintenance procedures θα κοινοποιούνται όσο το δυνατόν πιο άμεσα, και οι χρήστες που επηρεάζονται από τέτοια γεγονότα μπορεί να δικαιούνται credits όπως ορίζεται παρακάτω:

| Διάρκεια downtime    | Compensation                                                                                                                               |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Έως 6 ώρες            | Καθορίζεται κατά τη διακριτική ευχέρεια της ομάδας                                                                                         |
| Έως 12 ώρες           | Ισοδύναμο με 1 ημέρα service credit                                                                                                        |
| Έως 24 ώρες           | Ισοδύναμο με 2 ημέρες service credit                                                                                                       |
| Πάνω από 48 ώρες      | Πιστώνεται στο διπλάσιο της διάρκειας του downtime (για παράδειγμα, διακοπή 3 ημερών αποδίδει 6 ημέρες service credit)                    |

Η παρούσα πολιτική **εξαιρεί οποιοδήποτε downtime σχετίζεται με τις εφαρμογές που έχετε εγκαταστήσει.**

## Drive failure και απώλεια δεδομένων

Η πλατφόρμα Appbox βασίζεται σε shared infrastructure σχεδιασμένη να προσφέρει δίκαιη κατανομή πόρων και ολοκληρωμένη χρήση δίσκου. Παρά τις καλύτερες προσπάθειές μας, hardware failures, όπως drive malfunctions, μπορεί μερικές φορές να προκύψουν.

Αν drive failure σας προκαλέσει απώλεια πρόσβασης στα δεδομένα σας και αυτά δεν μπορούν να ανακτηθούν, θα δικαιούστε **έναν πλήρη μήνα υπηρεσίας χωρίς κόστος**.

Συνιστούμε έντονα να διατηρείτε backups όλων των βασικών δεδομένων που αποθηκεύετε στην υπηρεσία σας.

Παρακαλούμε σημειώστε ότι αν τα δεδομένα χαθούν λόγω ενεργειών που ξεκίνησε ο χρήστης (π.χ. τυχαία διαγραφή) και δεν εντοπιστεί system-related error, δεν θα παρέχεται compensation.

## Fair use των server resources

Οι Υπηρεσίες μας παρέχονται σε shared resources, οπότε είναι κρίσιμο κάθε πελάτης να διασφαλίζει ότι οι δραστηριότητές του δεν διαταράσσουν την απόδοση άλλων χρηστών και παραμένουν εντός αποδεκτών resource limits.

Το Appbox μπορεί, χωρίς προηγούμενη ειδοποίηση, να σταματήσει οποιεσδήποτε εφαρμογές ή processes που επηρεάζουν αρνητικά την εμπειρία άλλων πελατών. Σε σοβαρές περιπτώσεις resource misuse, η υπηρεσία του πελάτη μπορεί να ανασταλεί ή να τερματιστεί πλήρως.

### Disk I/O Fair Use Policy

Η παρούσα πολιτική εφαρμόζεται μόνο στα plans NG-8000 και μικρότερα (non-dedicated disk plans).

Για να διασφαλίσει βέλτιστη απόδοση για όλους τους χρήστες, το Appbox χρησιμοποιεί automated Disk I/O monitoring system που:

1. Παρακολουθεί συνεχώς τη disk utilization σε όλους τους shared resources
2. Εντοπίζει εφαρμογές που καταναλώνουν υπερβολικούς disk I/O resources για παρατεταμένα χρονικά διαστήματα
3. Εφαρμόζει αυτόματα throttling measures όταν είναι απαραίτητο

Αν η εφαρμογή σας εντοπιστεί να καταναλώνει υπερβολικό disk I/O:
- Μπορεί να λάβετε warning notification
- Η επίμονη υψηλή χρήση θα οδηγήσει σε automatic throttling της disk I/O speed σας
- Επαναλαμβανόμενες παραβιάσεις θα οδηγήσουν σε ολοένα αυστηρότερα throttling limits
- Το throttling είναι προσωρινό και θα αφαιρεθεί αυτόματα μόλις η χρήση επιστρέψει σε φυσιολογικά επίπεδα

Παρακαλούμε σημειώστε ότι resource-intensive applications μπορεί να ενεργοποιήσουν αυτά τα μέτρα αν εκτελούνται χωρίς μέριμνα για άλλους χρήστες.

Εφαρμόζουμε αυτή την πολιτική για να εγγυηθούμε ότι όλοι οι χρήστες έχουν ικανοποιητική εμπειρία στην πλατφόρμα μας.
