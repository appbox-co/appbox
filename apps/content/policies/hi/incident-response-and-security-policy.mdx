हमारी best efforts और rigorous audits के बावजूद, security incidents किसी भी online business का inevitable हिस्सा हैं। आपकी security हमारे लिए important है, और हम आपकी security और अपने platform दोनों की security के लिए पूरी तरह committed हैं। हम transparency और honesty की culture maintain करते हैं, जो security incidents report करने के हमारे approach तक भी extend होती है। Security incident की स्थिति में, Appbox निम्न actions लेगा:

1. Necessary होने पर, incident quickly resolve करने के लिए affected Appbox servers तक connections restrict करेंगे
2. जैसे ही कोई incidents report या discover होते हैं, हम अपने communication channels पर details publish करेंगे, जिनमें शामिल हैं:
   - हमारा website
   - हमारा [status page](https://status.appbox.co)
   - Affected clients को email notifications

हम affected clients को directly email करना भी शुरू करेंगे, हालांकि सभी communications dispatch होने में कुछ समय लग सकता है।

अगर आपको अपनी services में issues experience हो रहे हैं, तो कृपया हमारा [status page](https://status.appbox.co) check करें। सबसे likely आपको अपनी service से related maintenance work दिखेगा। अगर ongoing maintenance नहीं है, तो आप [support ticket खोलें](https://billing.appbox.co/submitticket.php?step=2&deptid=3) ताकि हम आपके issues investigate कर सकें और आपको accordingly guide कर सकें।

## Security-first infrastructure

* हम third-party partners और vendors को suitability और security risk दोनों के लिए carefully assess करते हैं। हम ensure करते हैं कि सभी partnerships के लिए सही technical और contractual commitments मौजूद हों।
* हम आपकी availability और security को ध्यान में रखते हुए enterprise-grade, high-availability data centers का उपयोग करते हैं—सभी rigorous physical security measures के साथ।
* हम customer data isolate करने, security enhance करने, और potential network-level intrusions prevent करने के लिए अपने network infrastructure में private VLANs implement करते हैं। हमारी in-house networking team optimal security सुनिश्चित करने के लिए इन VLANs को manage करती है।

## आप कैसे मदद कर सकते हैं

हमारी [Terms of Service](/policies/terms-of-service) और security documentation में described अनुसार, Appbox secure रहे यह सुनिश्चित करने के लिए हम extensive measures लेते हैं। आप platform को और protect करने में हमारी मदद कर सकते हैं:

* हमारी [Password Policy](/policies/password-policy) follow करके
* Account sharing में सावधानी रखकर—अपना account कभी उन लोगों से share न करें जिन्हें आप नहीं जानते या trust नहीं करते
* अगर आपको अपने account पर unauthorized access दिखे, जिसमें आपके password का कोई suspicious use भी शामिल है, तो [Appbox को immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) करके
* अपने account पर two-factor authentication (2FA) enable करके

## Security concerns report करना

* **Security Events**: Unauthorized account usage या suspected data breach जैसे security event report करने के लिए कृपया [Appbox को immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) करें
* **Vulnerabilities**: Vulnerabilities और potential rewards report करने की detailed information के लिए कृपया हमारा [Bug Bounty Program](/policies/bug-bounty-program) देखें
* **Sensitive Data**: अगर आपको Appbox से sensitive data remove करवाना है, तो कृपया हमारी [Sensitive Data Removal Policy](/policies/sensitive-data-removal-policy) देखें
* **Content Abuse**: Copyright या content abuse issues report करने का तरीका समझने के लिए हमारी [Content Abuse Policy](/policies/content-abuse-policy) पढ़ें
