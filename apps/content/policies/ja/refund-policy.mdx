## 7日間返金ポリシー

新規のお客様向けに、Appbox は7日間の返金期間を提供しています。最初の7日以内に当社サービスに完全には満足できない場合、[support ticket](https://billing.appbox.co/submitticket.php?step=2&deptid=3) を開くことで、日割り返金を request できます。

日割り返金は、サービスが使用された日数に基づいて計算されます。たとえば、Beginner plan を3日間使用してから返金を request した場合、4日分のサービス料金が返金されます。

## 返金対象

### 新規のお客様

上記のとおり、新規のお客様は、当社プラン（Beginner、Appbox、Dedicated）の初回購入から7日以内に request した場合、返金対象となります。

### 既存のお客様

既存のお客様には、以下のガイドラインが適用されます。

1. **Service Upgrades**: サービスをアップグレードして満足できない場合、downgrade を request できますが、差額の返金は行われません。
2. **Additional Services**: 追加サービス（例: dedicated IP addresses）について返金はありません。
3. **Service Renewals**: 初回購入から7日間が経過した後の service renewals について返金はありません。

### 返金制限

返金システムの abuse を防ぐため、お客様は **6か月間に1回の返金** に制限されます。つまり、過去6か月以内にすでに返金を受けている場合、前回の返金日から6か月が経過するまで、別の返金を受ける資格はありません。

## 返金不可の条件

以下の条件は、いかなる状況でも返金対象になりません。

1. Cryptocurrencies を使用した支払い
2. Dedicated IP addresses の購入
3. 3か月分のサービスを超える過払い

## 返金の request 方法

返金を request するには、以下の情報を含めて [support ticket](https://billing.appbox.co/submitticket.php?step=2&deptid=3) を作成してください。

1. Subject: Refund Request – [Your Username]
2. Message: 返金を request する理由と関連する詳細を含めてください

## キャンセルポリシー

当社 support ticket system から [cancellation request](https://billing.appbox.co/submitticket.php?step=2&deptid=4) を提出することで、いつでもサービスをキャンセルできます。

## 支払い方法ごとの詳細

### Credit Card / PayPal Payments

返金は10 business days 以内に元の支払い方法へ処理されます。

## 追加事項

- Abusive behavior、当社 [Terms of Service](/policies/terms-of-service) 違反、または fraudulent activity は、返金なしでアカウントを即時終了する結果となります。
- アカウントが終了されると、すべてのデータは永久に削除され、復元できません。
- 返金を request することは、アカウント終了およびデータ削除の受け入れを意味します。

当社 Refund Policy に関する質問は、[Support Ticket System](https://billing.appbox.co/submitticket.php?step=2&deptid=3) から support team へお問い合わせください。
