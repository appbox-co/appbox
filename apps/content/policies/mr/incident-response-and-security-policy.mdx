आमचे best efforts आणि rigorous audits असूनही, security incidents कोणत्याही online business चा अपरिहार्य भाग आहेत. तुमची security आमच्यासाठी महत्त्वाची आहे, आणि आम्ही तुमच्या security सोबत आमच्या platform security साठी पूर्णपणे committed आहोत. आम्ही transparency आणि honesty ची culture राखतो, आणि security incidents report करण्याच्या आमच्या approach मध्येही हे लागू होते. Security incident झाल्यास, Appbox खालील actions घेईल:

1. आवश्यक असल्यास, incident लवकर resolve करण्यासाठी affected Appbox servers कडे connections restrict करू
2. कोणतेही incidents report किंवा discover होताच, खालील communication channels वर details publish करू:
   - आमची website
   - आमचे [status page](https://status.appbox.co)
   - Affected clients ना email notifications

आम्ही affected clients ना direct emails पाठवायलाही सुरुवात करू, पण सर्व communications dispatch होण्यासाठी काही वेळ लागू शकतो.

तुमच्या services मध्ये issues येत असल्यास, कृपया आमचे [status page](https://status.appbox.co) तपासा. बहुधा तुम्हाला तुमच्या service शी संबंधित maintenance work दिसेल. Ongoing maintenance नसल्यास, तुमच्या issues तपासून तुम्हाला योग्य मार्गदर्शन करण्यासाठी [support ticket उघडू](https://billing.appbox.co/submitticket.php?step=2&deptid=3) शकता.

## Security-First Infrastructure

* आम्ही third-party partners आणि vendors चे suitability आणि security risk या दोन्हीसाठी काळजीपूर्वक assessment करतो. सर्व partnerships साठी योग्य technical आणि contractual commitments आहेत याची खात्री करतो.
* तुमची availability आणि security लक्षात घेऊन आम्ही enterprise-grade, high-availability data centers वापरतो - सर्व rigorous physical security measures सह.
* Customer data isolate करण्यासाठी, security enhance करण्यासाठी, आणि potential network-level intrusions टाळण्यासाठी आम्ही आमच्या network infrastructure मध्ये private VLANs implement करतो. Optimal security सुनिश्चित करण्यासाठी आमची in-house networking team हे VLANs manage करते.

## तुम्ही कशी मदत करू शकता

आमच्या [Terms of Service](/policies/terms-of-service) आणि security documentation मध्ये वर्णन केल्याप्रमाणे, Appbox secure राहावा यासाठी आम्ही extensive measures घेतो. Platform आणखी protect करण्यासाठी तुम्ही आम्हाला मदत करू शकता:

* आमची [Password Policy](/policies/password-policy) follow करून
* Account sharing बद्दल काळजी घेऊन - तुम्हाला माहिती नसलेल्या किंवा trust नसलेल्या लोकांसोबत account कधीही share करू नका
* तुमच्या account ला कोणताही unauthorized access दिसल्यास, तुमच्या password चा suspicious use समाविष्ट असल्यास, [Appbox ला लगेच notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) करून
* तुमच्या account वर two-factor authentication (2FA) enable करून

## Security Concerns Report करणे

* **Security Events**: Unauthorized account usage किंवा suspected data breach सारखा security event report करण्यासाठी कृपया [Appbox ला लगेच notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) करा
* **Vulnerabilities**: Vulnerabilities report करणे आणि potential rewards याबद्दल detailed information साठी कृपया आमचा [Bug Bounty Program](/policies/bug-bounty-program) पहा
* **Sensitive Data**: Appbox मधून sensitive data remove करायचा असल्यास, कृपया आमची [Sensitive Data Removal Policy](/policies/sensitive-data-removal-policy) पहा
* **Content Abuse**: Copyright किंवा content abuse issues कसे report करायचे ते समजण्यासाठी आमची [Content Abuse Policy](/policies/content-abuse-policy) वाचा
