మా best efforts మరియు rigorous audits ఉన్నప్పటికీ, security incidents ఏ online business లోనైనా unavoidable భాగం. మీ security మాకు ముఖ్యమైనది, మరియు మీ security తో పాటు మా platform security పట్ల మేము పూర్తిగా committed. Security incidents report చేయడంలో మా approach వరకు విస్తరించే transparency మరియు honesty culture ను మేము maintain చేస్తాము. Security incident జరిగినప్పుడు, Appbox క్రింది actions తీసుకుంటుంది:

1. అవసరమైతే, incident ను త్వరగా resolve చేయడానికి affected Appbox servers కు connections restrict చేస్తాము
2. ఏ incidents report లేదా discover అయిన వెంటనే, మా communication channels లో details publish చేస్తాము, వీటితో సహా:
   - మా website
   - మా [status page](https://status.appbox.co)
   - Affected clients కు email notifications

మేము affected clients కు directly email చేయడం కూడా ప్రారంభిస్తాము, అయితే అన్ని communications dispatch కావడానికి కొంత సమయం పట్టవచ్చు.

మీ services లో issues ఎదుర్కొంటున్నట్లయితే, దయచేసి మా [status page](https://status.appbox.co) చూడండి. మీ service కు సంబంధించిన maintenance work కనిపించే అవకాశం ఎక్కువ. Ongoing maintenance లేకపోతే, మేము మీ issues investigate చేసి accordingly guide చేయడానికి [support ticket open](https://billing.appbox.co/submitticket.php?step=2&deptid=3) చేయండి.

## Security-first infrastructure

* Suitability మరియు security risk రెండింటికీ third-party partners మరియు vendors ను మేము జాగ్రత్తగా assess చేస్తాము. అన్ని partnerships కోసం సరైన technical మరియు contractual commitments ఉండేలా ensure చేస్తాము.
* మీ availability మరియు security దృష్టిలో ఉంచుకుని enterprise-grade, high-availability data centers ఉపయోగిస్తాము, అన్నీ rigorous physical security measures తో.
* Customer data isolate చేయడానికి, security enhance చేయడానికి, potential network-level intrusions prevent చేయడానికి మా network infrastructure లో private VLANs అమలు చేస్తాము. Optimal security ensure చేయడానికి మా in-house networking team ఈ VLANs manage చేస్తుంది.

## మీరు ఎలా సహాయపడగలరు

మా [Terms of Service](/policies/terms-of-service) మరియు మా security documentation అంతటా వివరించినట్లుగా, Appbox secure గా ఉండేలా మేము extensive measures తీసుకుంటాము. Platform ను మరింత protect చేయడంలో మీరు సహాయపడగల మార్గాలు:

* మా [Password Policy](/policies/password-policy) పాటించడం
* Account sharing గురించి జాగ్రత్తగా ఉండడం—మీకు తెలియని లేదా trust లేని వ్యక్తులతో మీ account ఎప్పుడూ share చేయకండి
* మీ password యొక్క suspicious use సహా మీ account కు unauthorized access గమనిస్తే [Appbox కు వెంటనే notify చేయడం](https://billing.appbox.co/submitticket.php?step=2&deptid=3)
* మీ account పై two-factor authentication (2FA) enable చేయడం

## Security concerns report చేయడం

* **Security Events**: Unauthorized account usage లేదా suspected data breach వంటి security event report చేయడానికి దయచేసి [Appbox కు వెంటనే notify చేయండి](https://billing.appbox.co/submitticket.php?step=2&deptid=3)
* **Vulnerabilities**: Vulnerabilities మరియు potential rewards report చేయడంపై detailed information కోసం మా [Bug Bounty Program](/policies/bug-bounty-program) చూడండి
* **Sensitive Data**: Appbox నుండి sensitive data remove చేయించుకోవాల్సి ఉంటే, మా [Sensitive Data Removal Policy](/policies/sensitive-data-removal-policy) చూడండి
* **Content Abuse**: Copyright లేదా content abuse issues report చేయడం ఎలా అన్నది అర్థం చేసుకోవడానికి మా [Content Abuse Policy](/policies/content-abuse-policy) చదవండి
