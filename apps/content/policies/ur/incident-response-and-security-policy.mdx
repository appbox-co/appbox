ہماری best efforts اور rigorous audits کے باوجود، security incidents کسی بھی online business کا inevitable part ہیں۔ آپ کی security ہمارے لیے اہم ہے، اور ہم آپ کی security اور اپنے platform دونوں کی security کے لیے fully committed ہیں۔ ہم transparency اور honesty کی culture maintain کرتے ہیں، جو security incidents report کرنے کے ہمارے approach تک extend ہوتی ہے۔ security incident کی صورت میں، Appbox درج ذیل actions لے گا:

1. اگر necessary ہو، ہم incident کو quickly resolve کرنے کے لیے affected Appbox servers تک connections restrict کریں گے
2. جیسے ہی کوئی incidents report یا discover ہوں، ہم اپنے communication channels پر details publish کریں گے، including:
   - ہماری website
   - ہمارا [status page](https://status.appbox.co)
   - affected clients کو email notifications

ہم affected clients کو directly email کرنا بھی start کریں گے، اگرچہ تمام communications dispatch ہونے میں کچھ time لگ سکتا ہے۔

اگر آپ اپنی services کے ساتھ issues experience کر رہے ہیں، please ہمارا [status page](https://status.appbox.co) check کریں۔ غالب امکان ہے کہ آپ اپنی service سے related maintenance work دیکھیں گے۔ اگر ongoing maintenance نہیں ہے، تو بلا جھجھک [support ticket open](https://billing.appbox.co/submitticket.php?step=2&deptid=3) کریں تاکہ ہم آپ کے issues investigate کر سکیں اور accordingly guide کر سکیں۔

## Security-first infrastructure

* ہم third-party partners اور vendors کو suitability اور security risk دونوں کے لیے carefully assess کرتے ہیں۔ ہم ensure کرتے ہیں کہ تمام partnerships کے لیے right technical اور contractual commitments موجود ہوں۔
* ہم enterprise-grade، high-availability data centers use کرتے ہیں جو آپ کی availability اور security کو مدنظر رکھتے ہیں، سب rigorous physical security measures کے ساتھ۔
* ہم اپنے network infrastructure کے اندر private VLANs implement کرتے ہیں تاکہ customer data isolate ہو، security enhance ہو، اور potential network-level intrusions prevent ہوں۔ ہماری in-house networking team optimal security ensure کرنے کے لیے ان VLANs کو manage کرتی ہے۔

## آپ کیسے مدد کر سکتے ہیں

جیسا کہ ہمارے [Terms of Service](/policies/terms-of-service) اور security documentation میں describe کیا گیا ہے، ہم Appbox کو secure ensure کرنے کے لیے extensive measures لیتے ہیں۔ آپ platform کو مزید protect کرنے میں ہماری مدد کر سکتے ہیں:

* ہماری [Password Policy](/policies/password-policy) follow کر کے
* account sharing کے بارے میں careful رہ کر، اپنا account ان لوگوں کے ساتھ کبھی share نہ کریں جنہیں آپ نہیں جانتے یا trust نہیں کرتے
* اگر آپ اپنے account تک unauthorized access notice کریں، including اپنے password کا کوئی suspicious use، تو [Appbox کو immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) کر کے
* اپنے account پر two-factor authentication (2FA) enable کر کے

## Security concerns report کرنا

* **Security Events**: security event، جیسے unauthorized account usage یا suspected data breach، report کرنے کے لیے please [Appbox کو immediately notify](https://billing.appbox.co/submitticket.php?step=2&deptid=3) کریں
* **Vulnerabilities**: vulnerabilities report کرنے اور potential rewards کے بارے میں detailed information کے لیے please ہمارا [Bug Bounty Program](/policies/bug-bounty-program) دیکھیں
* **Sensitive Data**: اگر آپ کو Appbox سے sensitive data remove کرانا ہے، ہماری [Sensitive Data Removal Policy](/policies/sensitive-data-removal-policy) refer کریں
* **Content Abuse**: copyright یا content abuse issues report کرنے کا طریقہ سمجھنے کے لیے ہماری [Content Abuse Policy](/policies/content-abuse-policy) پڑھیں
