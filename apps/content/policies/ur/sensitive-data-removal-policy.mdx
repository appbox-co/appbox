## Sensitive information کو سمجھنا

Appbox میں، ہم sensitive information کو یوں define کرتے ہیں:

1. Personal contact details
2. Identification numbers and documents
3. Financial information
4. Authentication credentials
5. کوئی بھی other information جو expose ہونے پر risk pose کر سکتی ہو

## Data deletion کا آپ کا حق

UK GDPR اور Data Protection Act 2018 کے مطابق، آپ کو اپنے personal data کی deletion request کرنے کا right ہے ("right to be forgotten")۔ جب آپ ایسی request کرتے ہیں:

1. ہم آپ کا account اور associated personal information delete کریں گے
2. ہم آپ کا data اپنے active systems سے remove کریں گے
3. آپ کا data Appbox staff یا other users کے لیے accessible نہیں رہے گا

## Legal compliance کے لیے data retention

اگرچہ ہم آپ کا account اور personal information delete کر سکتے ہیں، ہمیں UK tax regulations اور HMRC requirements comply کرنے کے لیے legally certain transaction records retain کرنا required ہے:

1. Transaction records including dates، amounts، اور basic identification information
2. Invoice details اور payment confirmations
3. tax reporting اور financial audits کے لیے necessary کوئی بھی information

یہ retained information:
- صرف اتنی مدت تک رکھی جاتی ہے جتنی legally required ہو
- appropriate safeguards کے ساتھ secured ہوتی ہے
- صرف ان personnel کے لیے accessible ہوتی ہے جنہیں compliance purposes کے لیے اس کی ضرورت ہو
- marketing یا other commercial purposes کے لیے use نہیں ہوتی

## Username modifications

اگرچہ ہم سمجھتے ہیں کہ circumstances آپ کو username change کرنے کی ضرورت دے سکتی ہیں، ہماری technical architecture میں certain limitations ہیں۔ ہمارے system کے design کی وجہ سے، ہم existing Appbox کو simply rename نہیں کر سکتے۔ اس کے بجائے، ہمارا process شامل ہے:

1. آپ کے requested username کے ساتھ new Appbox create کرنا
2. آپ کو اپنا data transfer کرنے کے options provide کرنا:
   - اپنے content کی self-migration new Appbox پر
   - support-assisted migration (upon request)
3. successful data transfer کے بعد آپ کا previous Appbox permanently delete کرنا

Username changes کے لیے ہمیں valid justification require ہے تاکہ اس service کے abuse کو prevent کیا جا سکے۔

## Email address updates

اپنی contact information current رکھنا important ہے۔ اپنے account سے associated email address update کرنے کے لیے:

1. اپنی desired new email address specify کرتے ہوئے ہمارے [Billing & Account Questions](https://billing.appbox.co/submitticket.php?step=2&deptid=2) department کے ذریعے support ticket submit کریں
2. verification کے بعد، ہم آپ کے account records update کریں گے
3. اس کے بعد آپ future logins اور communications کے لیے یہی new email address use کریں گے

ہم strongly recommend کرتے ہیں کہ up-to-date email address maintain کریں، کیونکہ ہم critical service notifications، maintenance alerts، اور account communications کے لیے یہ channel use کرتے ہیں۔

## Data removal request کرنے کا طریقہ

کسی بھی sensitive data removal process initiate کرنے کے لیے:

1. اپنے [Appbox Client Area](https://billing.appbox.co) میں log in کریں
2. ہمارے [Support Ticket System](https://billing.appbox.co/submitticket.php?step=2&deptid=3) کے ذریعے detailed ticket create کریں
3. اپنی request کی nature clearly state کریں اور کوئی necessary information provide کریں
4. ہماری team آپ کی request review کرے گی اور appropriate next steps کے ساتھ respond کرے گی

ہم اپنے users کی privacy prioritize کرتے ہیں اور ایسی تمام requests کو promptly اور discretion کے ساتھ handle کریں گے۔
